# 前端学习指南

> 全面系统的 JavaScript、HTML、CSS 学习文档站点

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run docs:dev
```

访问 http://localhost:5173 查看文档站点

### 构建生产版本

```bash
npm run docs:build
```

### 预览生产版本

```bash
npm run docs:preview
```

## ✨ 特性

- 🎨 **美观的界面** - 现代化的设计风格，紫色调主题
- 🔍 **全文搜索** - 内置本地搜索功能，快速定位知识点
- 🌙 **深色模式** - 自动适配系统主题，支持手动切换
- 📱 **响应式设计** - 完美适配桌面端和移动端
- 📖 **目录导航** - 清晰的侧边栏和文档目录
- 🌐 **中文优化** - 完全中文化的界面和内容

## 🛠️ 技术栈

- [VitePress](https://vitepress.dev/) - Vue 驱动的静态站点生成器
- [Vue 3](https://vuejs.org/) - 渐进式 JavaScript 框架
- Markdown - 文档编写
