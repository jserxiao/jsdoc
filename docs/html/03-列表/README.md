# 三、列表

## 无序列表

```html
<ul>
    <li>项目一</li>
    <li>项目二</li>
    <li>项目三</li>
</ul>

<!-- 嵌套列表 -->
<ul>
    <li>项目一
        <ul>
            <li>子项目一</li>
            <li>子项目二</li>
        </ul>
    </li>
</ul>
```

---

## 有序列表

```html
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>

<!-- 指定起始值 -->
<ol start="5">
    <li>从5开始</li>
</ol>

<!-- 反序 -->
<ol reversed>
    <li>倒序排列</li>
</ol>
```

---

## 定义列表

```html
<dl>
    <dt>术语一</dt>
    <dd>定义描述一</dd>
    
    <dt>术语二</dt>
    <dd>定义描述二</dd>
</dl>
```
