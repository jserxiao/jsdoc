# 二、文本内容

## 标题与段落

```html
<!-- 标题 h1-h6 -->
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>

<!-- 段落 -->
<p>这是一个段落。</p>

<!-- 换行 -->
<p>第一行<br>第二行</p>

<!-- 水平线 -->
<hr>
```

---

## 文本格式化

```html
<strong>重要文本</strong>  <!-- 粗体，语义化 -->
<b>粗体文本</b>           <!-- 粗体，无语义 -->
<em>强调文本</em>         <!-- 斜体，语义化 -->
<i>斜体文本</i>           <!-- 斜体，无语义 -->
<mark>高亮文本</mark>
<del>删除文本</del>
<ins>插入文本</ins>
<sub>下标</sub>
<sup>上标</sup>
<small>小号文本</small>
```

---

## 引用与代码

```html
<!-- 引用 -->
<blockquote cite="url">
    长引用内容
</blockquote>

<q cite="url">短引用</q>

<cite>引用来源</cite>

<!-- 代码 -->
<code>inline code</code>
<pre>preformatted text</pre>
<kbd>键盘输入</kbd>
<samp>示例输出</samp>
<var>变量</var>
```
