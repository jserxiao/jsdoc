# Turbopack

Turbopack 是用 Rust 编写的增量打包器，专为大型应用优化，是 Webpack 的下一代继任者。

---

## 概述

| 特性 | 说明 |
|------|------|
| 定位 | 增量打包器 |
| 作者 | Vercel |
| 开发语言 | Rust |
| 核心优势 | 增量编译、持久缓存 |
| 适用场景 | Next.js 大型应用 |

---

## 快速开始

```bash
# 创建 Next.js 项目
npx create-next-app@latest --turbo

# 或在现有项目中启用
next dev --turbo
```

---

## 适用场景

- ✅ Next.js 项目
- ✅ 大型应用增量构建
- ✅ 追求开发效率
- ✅ 需要 Webpack 生态兼容

---

## 与 Vite 对比

| 特性 | Turbopack | Vite |
|------|-----------|------|
| 开发语言 | Rust | Go + JS |
| 冷启动 | 极快 | 极快 |
| HMR | 更快 | 快 |
| 生产构建 | 待完善 | Rollup |
| 生态绑定 | Next.js | 通用 |
| 持久缓存 | 内存缓存 | 文件缓存 |
