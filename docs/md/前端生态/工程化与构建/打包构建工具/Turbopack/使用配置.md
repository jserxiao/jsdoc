# 使用配置

## Next.js 集成

### 启用 Turbopack

```javascript
// next.config.js
module.exports = {
  experimental: {
    turbo: {
      enabled: true
    }
  }
}
```

```bash
# 开发模式使用 Turbopack
next dev --turbo
```

### 配置选项

```javascript
// next.config.js
module.exports = {
  experimental: {
    turbo: {
      // 模块解析配置
      resolveExtensions: ['.tsx', '.ts', '.jsx', '.js', '.json'],
      
      // 路径别名
      resolveAlias: {
        '@': './src',
        '@components': './src/components',
        '@utils': './src/utils'
      },
      
      // 自定义 loader（实验性）
      loaders: {
        '.md': ['@mdx-js/loader']
      }
    }
  }
}
```

## 完整配置示例

```javascript
// next.config.js
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    turbo: {
      resolveExtensions: [
        '.tsx',
        '.ts',
        '.jsx',
        '.js',
        '.json',
        '.css',
        '.scss'
      ],
      resolveAlias: {
        '@': './src',
        '@components': './src/components',
        '@hooks': './src/hooks',
        '@utils': './src/utils',
        '@styles': './src/styles',
        '@types': './src/types',
        '@lib': './src/lib'
      }
    }
  },
  
  // 其他 Next.js 配置
  reactStrictMode: true,
  swcMinify: true
}

module.exports = nextConfig
```

## 支持的文件类型

### JavaScript/TypeScript

```typescript
// 直接支持
import Component from './Component'
import { hook } from './hooks'
import type { Props } from './types'
```

### CSS

```javascript
// CSS Modules
import styles from './Component.module.css'

// 全局 CSS
import './globals.css'

// Sass（需安装 sass）
import styles from './Component.module.scss'
```

### 静态资源

```javascript
// 图片
import logo from './logo.png'
import Icon from './icon.svg'

// 字体
import font from './font.woff2'

// JSON
import pkg from './package.json'
```

## 从 Webpack 迁移

### 兼容的配置

```javascript
// Webpack 配置
{
  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src')
    },
    extensions: ['.js', '.ts', '.jsx', '.tsx']
  }
}

// 对应的 Turbopack 配置
{
  experimental: {
    turbo: {
      resolveAlias: {
        '@': './src'
      },
      resolveExtensions: ['.js', '.ts', '.jsx', '.tsx']
    }
  }
}
```

### 不兼容的功能

- 自定义 Webpack Loader（部分支持）
- 自定义 Webpack Plugin
- 复杂的 Webpack 配置

## 限制

### 当前限制

- 仅支持 Next.js 13+
- 开发环境可用，生产构建仍使用 Webpack
- 部分自定义 Loader 可能不兼容

### 未来规划

1. **独立使用**：脱离 Next.js
2. **插件系统**：支持自定义扩展
3. **生产构建**：替代 Webpack
4. **更多框架**：支持 React、Vue 等

## 故障排除

### 清除缓存

```bash
# 清除 Turbopack 缓存
rm -rf .turbo

# 或使用 Next.js 清除
rm -rf .next
```

### 常见问题

```javascript
// 问题：路径别名不生效
// 解决：确保 resolveAlias 使用相对路径
{
  turbo: {
    resolveAlias: {
      '@': './src'  // 使用相对路径
    }
  }
}

// 问题：CSS Modules 类名冲突
// 解决：配置 localIdentName
{
  turbo: {
    // 暂不支持自定义，使用默认配置
  }
}
```
