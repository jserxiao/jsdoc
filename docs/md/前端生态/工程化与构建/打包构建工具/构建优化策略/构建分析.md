## Webpack Bundle Analyzer

### 安装使用

```bash
npm install --save-dev webpack-bundle-analyzer
```

```javascript
const { BundleAnalyzerPlugin } = require('webpack-bundle-analyzer')

module.exports = {
  plugins: [
    new BundleAnalyzerPlugin({
      analyzerMode: 'static',
      openAnalyzer: false,
      reportFilename: 'report.html'
    })
  ]
}
```

### 命令行使用

```bash
# 生成构建分析
webpack --profile --json > stats.json

# 分析
webpack-bundle-analyzer stats.json
```

## 自定义分析脚本

```javascript
const webpack = require('webpack')
const config = require('./webpack.config')

const compiler = webpack(config)

compiler.run((err, stats) => {
  if (err) {
    console.error(err)
    return
  }
  
  const info = stats.toJson({
    assets: true,
    modules: true,
    chunks: true
  })
  
  // 输出资源大小
  console.log('Assets:')
  info.assets.forEach(asset => {
    console.log(`  ${asset.name}: ${(asset.size / 1024).toFixed(2)} KB`)
  })
  
  // 输出模块数量
  console.log(`\nModules: ${info.modules.length}`)
  
  // 输出 Chunk 数量
  console.log(`Chunks: ${info.chunks.length}`)
  
  // 输出警告和错误
  if (info.warnings.length) {
    console.log('\nWarnings:', info.warnings)
  }
  
  if (info.errors.length) {
    console.log('\nErrors:', info.errors)
  }
})
```

## 持续监控

### CI/CD 集成

```yaml
# .github/workflows/build.yml
name: Build Analysis

on: [push]

jobs:
  analyze:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Install
        run: npm install
      
      - name: Build
        run: npm run build
      
      - name: Analyze
        run: |
          npm run build:analyze
          
      - name: Upload Report
        uses: actions/upload-artifact@v4
        with:
          name: build-report
          path: report.html
```

### 构建产物对比

```javascript
const fs = require('fs')

function compareBuilds(oldStats, newStats) {
  const oldAssets = require(oldStats).assets
  const newAssets = require(newStats).assets
  
  const comparison = {}
  
  oldAssets.forEach(asset => {
    comparison[asset.name] = {
      old: asset.size,
      new: null,
      diff: null
    }
  })
  
  newAssets.forEach(asset => {
    if (comparison[asset.name]) {
      comparison[asset.name].new = asset.size
      comparison[asset.name].diff = asset.size - comparison[asset.name].old
    } else {
      comparison[asset.name] = {
        old: null,
        new: asset.size,
        diff: null
      }
    }
  })
  
  return comparison
}
```

## 分析报告解读

### 重点关注

1. **重复依赖**：同一包的多个版本
2. **大型依赖**：体积过大的包
3. **未使用代码**：Tree Shaking 未生效
4. **同步导入**：可改为异步的模块

### 优化建议

```javascript
// 发现大型依赖后
// 1. 寻找轻量替代品
// lodash → lodash-es
// moment → dayjs

// 2. 按需导入
import { Button } from 'antd'

// 3. 动态导入
const Heavy = () => import('./Heavy')
```
