# 构建优化策略

构建优化是前端工程化的重要组成部分，涵盖速度优化和产物优化两个方面。

---

## 概述

| 优化方向 | 策略 | 效果 |
|----------|------|------|
| **速度优化** | 持久化缓存、并行构建、SWC/esbuild | 构建速度提升 5-10x |
| **产物优化** | 代码分割、Tree Shaking、压缩 | 体积减少 30-50% |

## 快速诊断

### 构建速度慢

1. 开启持久化缓存
2. 使用 SWC/esbuild 替代 Babel
3. 并行构建
4. 减少不必要的编译范围

### 包体积过大

1. 分析依赖树
2. 按需引入，避免全量导入
3. 代码分割，动态加载
4. 使用更轻量的替代方案
