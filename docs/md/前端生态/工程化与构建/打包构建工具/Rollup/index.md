# Rollup

Rollup 是专注于库打包的构建工具，输出更加干净，Tree Shaking 效果最佳。

---

## 概述

| 特性 | 说明 |
|------|------|
| 定位 | 库打包工具 |
| 作者 | Rich Harris（Svelte 作者） |
| 核心优势 | Tree Shaking、输出干净 |
| 输出格式 | ESM、CommonJS、UMD、IIFE |
| 适用场景 | 库开发、组件库 |

## 快速开始

```bash
# 安装
npm install rollup --save-dev

# 构建
npx rollup -c
```

---

## 适用场景

- ✅ 库开发（UI 组件库、工具库）
- ✅ 需要多种输出格式
- ✅ 追求输出代码精简
- ✅ 简单应用打包

---

## 与 Webpack 对比

| 特性 | Rollup | Webpack |
|------|--------|---------|
| Tree Shaking | 原生支持，效果最佳 | 需要配置 |
| 输出体积 | 更小 | 较大（有运行时） |
| 代码分割 | 基础支持 | 强大 |
| 生态 | 库开发为主 | 应用开发为主 |
| 学习曲线 | 简单 | 复杂 |
