# Yarn

## 简介

Yarn 是 Facebook 开发的新一代包管理器，解决了 npm 早期版本的问题：
- 并行安装，速度更快
- 离线缓存机制
- 更好的安全性

## 安装

```bash
# 通过 npm 安装
npm install -g yarn

# 查看版本
yarn --version
```

## 基本命令

### 项目初始化

```bash
# 创建 package.json
yarn init
yarn init -y
```

### 安装依赖

```bash
# 安装所有依赖
yarn
yarn install

# 安装生产依赖
yarn add <package>

# 安装开发依赖
yarn add <package> --dev
yarn add <package> -D

# 安装全局包
yarn global add <package>

# 安装特定版本
yarn add <package>@1.2.3
```

### 卸载依赖

```bash
yarn remove <package>

# 卸载全局包
yarn global remove <package>
```

### 更新依赖

```bash
# 更新依赖
yarn upgrade
yarn upgrade <package>

# 交互式更新
yarn upgrade-interactive
```

### 运行脚本

```bash
# 运行脚本
yarn <script>
yarn build
yarn test
yarn start

# 传递参数
yarn build --mode production
```

### 其他命令

```bash
# 查看包信息
yarn info <package>

# 清理缓存
yarn cache clean

# 检查安全漏洞
yarn audit

# 查看全局安装路径
yarn global dir
```

## Yarn 工作原理

### yarn.lock

类似于 package-lock.json，锁定依赖版本：

```
# THIS IS AN AUTOGENERATED FILE. DO NOT EDIT THIS FILE DIRECTLY.
# yarn lockfile v1

lodash@^4.17.21:
  version "4.17.21"
  resolved "https://registry.yarnpkg.com/lodash/-/lodash-4.17.21.tgz"
  integrity sha512-v2kDEe57lecTulaDIuNTPy3Ry4gLGJ6Z1O3vE1krgXZNrsQ+LFTGHVxVjcXPs17LhbZVGedAJv8XZ1tvj5FvSg==
```

### Plug'n'Play (PnP)

Yarn v2+ 引入的新机制：
- 不再需要 node_modules 目录
- 使用 .pnp.cjs 文件管理依赖映射
- 启动速度更快，磁盘占用更少

```bash
# 启用 Yarn 2
yarn set version berry

# 启用 PnP
yarn config set nodeLinker pnp

# 启用 node_modules 模式
yarn config set nodeLinker node-modules
```

### .yarnrc.yml 配置

```yaml
yarnPath: .yarn/releases/yarn-4.0.0.cjs
nodeLinker: pnp

packageExtensions:
  "styled-components@*":
    dependencies:
      react-is: "*"

pnpFallbackMode: all
```
