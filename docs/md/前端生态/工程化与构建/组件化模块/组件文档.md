# 组件文档

组件文档是组件库成功的关键因素。良好的文档能够帮助开发者快速理解和使用组件，降低学习成本，提高开发效率。本章介绍主流的组件文档工具和最佳实践。

---

## 文档工具概览

| 工具 | 适用场景 | 框架支持 | 特点 |
|------|----------|----------|------|
| Storybook | 组件开发与文档 | React/Vue/Angular/Svelte | 最流行、功能丰富、插件生态 |
| VitePress | 文档站点 | 框架无关 | 轻量、快速、适合文档站 |
| Dumi | 组件库文档 | React | 蚂蚁出品、内置 API 提取 |
| Volar/vue-docgen-api | 类型提示 | Vue | IDE 集成、自动补全 |
| TypeDoc | API 文档 | TypeScript | 从代码生成 API 文档 |

---

## 一、Storybook

Storybook 是最流行的组件开发环境，支持独立开发、测试和文档化 UI 组件。

### 安装与初始化

```bash
# 在现有项目中添加
npx storybook@latest init

# 启动开发服务器
npm run storybook

# 构建静态站点
npm run build-storybook
```

### Story 文件结构

```typescript
// Button.stories.ts
import type { Meta, StoryObj } from '@storybook/vue3'
import Button from './Button.vue'

/**
 * Button 组件用于触发操作。
 * 
 * 支持多种变体和尺寸，可自定义图标和加载状态。
 */
const meta: Meta<typeof Button> = {
  title: 'Components/Button',
  component: Button,
  
  // 自动生成文档
  tags: ['autodocs'],
  
  // 参数控制
  argTypes: {
    variant: {
      control: 'select',
      options: ['default', 'primary', 'success', 'warning', 'danger'],
      description: '按钮类型',
      table: {
        defaultValue: { summary: 'default' },
      },
    },
    size: {
      control: 'select',
      options: ['small', 'medium', 'large'],
      description: '按钮尺寸',
    },
    disabled: {
      control: 'boolean',
      description: '是否禁用',
    },
    loading: {
      control: 'boolean',
      description: '是否显示加载状态',
    },
    onClick: {
      action: 'clicked',
      description: '点击事件',
    },
  },
  
  // 默认参数
  args: {
    variant: 'default',
    size: 'medium',
    disabled: false,
    loading: false,
  },
}

export default meta
type Story = StoryObj<typeof Button>

/**
 * 基础按钮用法
 */
export const Primary: Story = {
  args: {
    variant: 'primary',
    default: 'Primary Button',
  },
}

/**
 * 禁用状态
 */
export const Disabled: Story = {
  args: {
    variant: 'primary',
    disabled: true,
    default: 'Disabled Button',
  },
}

/**
 * 加载状态
 */
export const Loading: Story = {
  args: {
    variant: 'primary',
    loading: true,
    default: 'Loading...',
  },
}

/**
 * 不同尺寸
 */
export const Sizes: Story = {
  render: () => ({
    components: { Button },
    template: `
      <div style="display: flex; gap: 8px; align-items: center;">
        <Button size="small">Small</Button>
        <Button size="medium">Medium</Button>
        <Button size="large">Large</Button>
      </div>
    `,
  }),
}

/**
 * 所有变体
 */
export const AllVariants: Story = {
  render: () => ({
    components: { Button },
    template: `
      <div style="display: flex; gap: 8px; flex-wrap: wrap;">
        <Button variant="default">Default</Button>
        <Button variant="primary">Primary</Button>

    `,
  }),
}

## 设计指南

### 按钮层级

1. **主要按钮**：页面中最重要的操作
2. **次要按钮**：次要操作
3. **文字按钮**：最低优先级操作

### 最佳实践

✅ **推荐**
- 使用清晰的动词描述操作
- 按钮文案简洁有力
- 适当使用图标增强语义

❌ **不推荐**
- 在按钮中使用长句子
- 在同一区域放置多个主要按钮
- 禁用按钮不提供说明
```

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| variant | 按钮类型 | `'default' \| 'primary' \| 'success' \| 'warning' \| 'danger'` | `'default'` |

| loading | 是否加载中 | `boolean` | `false` |
| icon | 图标名称 | `string` | `-` |
| icon-position | 图标位置 | `'left' \| 'right'` | `'left'` |
| block | 是否为块级按钮 | `boolean` | `false` |

### Events

| 事件 | 说明 | 回调参数 |
|------|------|----------|
| click | 点击按钮时触发 | `(event: MouseEvent) => void` |

### Slots

| 插槽 | 说明 |
|------|------|
| default | 按钮内容 |
| icon | 自定义图标 |

### 自定义 Demo 插件

```typescript
// docs/.vitepress/plugins/demo.ts
import MarkdownIt from 'markdown-it'
import { render } from './demo-renderer'

export function demoPlugin(md: MarkdownIt) {
  md.inline.ruler.before('emphasis', 'demo', (state, silent) => {
    const match = /<demo\s+src=["']([^"']+)["']\s*\/?>/.exec(state.src.slice(state.pos))
    if (!match) return false
    
    if (silent) return true
    
    const src = match[1]
    const token = state.push('demo', 'demo', 0)
    token.meta = { src }
    token.content = match[0]
    
    state.pos += match[0].length
    return true
  })
  
  md.renderer.rules.demo = (tokens, idx) => {
    const { src } = tokens[idx].meta
    return render(src)
  }
}

// 使用
// .vitepress/config.ts
export default defineConfig({
  markdown: {
    config: (md) => {
      md.use(demoPlugin)
    },
  },
})

    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'
      
      - name: Install
        run: pnpm install
      
      - name: Build Storybook
        run: pnpm build-storybook
      
      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./storybook-static
```
