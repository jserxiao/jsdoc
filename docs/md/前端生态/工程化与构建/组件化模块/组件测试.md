# 组件测试

## 测试策略

| 测试类型 | 说明 | 工具 |
|----------|------|------|
| 单元测试 | 测试组件逻辑 | Vitest / Jest |
| 组件测试 | 测试组件渲染和交互 | Vue Test Utils |
| 快照测试 | 检测 UI 变化 | Vitest |
| E2E 测试 | 测试用户流程 | Cypress / Playwright |

## Vue Test Utils

### 安装

```bash
npm install -D @vue/test-utils vitest jsdom
```

### 配置

```typescript
// vitest.config.ts
import { defineConfig } from 'vitest/config'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  test: {
    environment: 'jsdom',
    globals: true
  }
})
```

### 基础测试

```typescript
// Button.test.ts
import { describe, it, expect } from 'vitest'
import { mount } from '@vue/test-utils'
import Button from './Button.vue'

describe('Button', () => {
  it('renders properly', () => {
    const wrapper = mount(Button, {
      slots: {
        default: 'Click me'
      }
    })
    expect(wrapper.text()).toContain('Click me')
  })

  it('emits click event', async () => {
    const wrapper = mount(Button)
    await wrapper.trigger('click')
    expect(wrapper.emitted()).toHaveProperty('click')
  })

  it('should not emit click when disabled', async () => {
    const wrapper = mount(Button, {
      props: { disabled: true }
    })
    await wrapper.trigger('click')
    expect(wrapper.emitted()).not.toHaveProperty('click')
  })

  it('applies type class correctly', () => {
    const wrapper = mount(Button, {
      props: { type: 'primary' }
    })
    expect(wrapper.classes()).toContain('my-button--primary')
  })
})
```

### Props 测试

```typescript
it('renders with default props', () => {
  const wrapper = mount(Button)
  expect(wrapper.props('type')).toBe('default')
  expect(wrapper.props('size')).toBe('medium')
})

it('accepts custom props', () => {
  const wrapper = mount(Button, {
    props: {
      type: 'primary',
      size: 'large'
    }
  })
  expect(wrapper.props('type')).toBe('primary')
  expect(wrapper.classes()).toContain('my-button--large')
})
```

### 插槽测试

```typescript
it('renders slot content', () => {
  const wrapper = mount(Button, {
    slots: {
      default: '<span class="icon">🔍</span>Search'
    }
  })
  expect(wrapper.find('.icon').exists()).toBe(true)
  expect(wrapper.text()).toContain('Search')
})

it('renders named slots', () => {
  const wrapper = mount(Card, {
    slots: {
      header: 'Card Title',
      default: 'Card Content',
      footer: 'Card Footer'
    }
  })
  expect(wrapper.find('.card-header').text()).toBe('Card Title')
})
```

### 事件测试

```typescript
it('emits custom events with payload', async () => {
  const wrapper = mount(SearchInput)

  await wrapper.find('input').setValue('hello')
  await wrapper.find('form').trigger('submit')

  expect(wrapper.emitted('submit')).toBeTruthy()
  expect(wrapper.emitted('submit')[0]).toEqual(['hello'])

```

### 异步测试

```typescript
it('handles async data', async () => {
  const wrapper = mount(UserList, {
    global: {
```

## 快照测试

```typescript
import { describe, it, expect } from 'vitest'
import { mount } from '@vue/test-utils'
import Button from './Button.vue'

describe('Button snapshots', () => {
  it('matches primary button snapshot', () => {
    const wrapper = mount(Button, {
      props: { type: 'primary' },
      slots: { default: 'Primary' }
    })
    expect(wrapper.html()).toMatchSnapshot()
  })
})
```

## 测试最佳实践

1. **测试行为而非实现**：关注用户如何使用组件
2. **保持测试独立**：每个测试应该独立运行
3. **使用 data-testid**：为测试添加选择器
4. **Mock 外部依赖**：API、路由、Store 等
5. **合理的覆盖率**：核心组件 80% 以上
