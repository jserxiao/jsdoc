# 开发 CLI

> 自定义 CLI 工具开发是前端工程化的重要技能，掌握 CLI 开发流程可以帮助团队构建符合自身需求的脚手架工具。

## 模块导航

| 模块 | 说明 |
|------|------|
| [开发流程](./开发流程/index.md) | CLI 完整开发流程：项目初始化、命令解析、交互设计、模板处理、发布维护 |
| [工具库详解](./工具库详解/index.md) | 核心工具库详解：Commander、Inquirer、Chalk、Ora、fs-extra 等 |
| [实战项目 xcli](./实战项目xcli/index.md) | 以 @jserxiao/xcli 为例，深入分析企业级 CLI 架构设计与核心实现 |

```
入门 → 进阶 → 实战

2. 进阶阶段
   ├── 掌握交互式问答设计

   ├── 分析优秀开源项目
```
