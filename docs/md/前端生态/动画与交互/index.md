# 动画与交互

## 概述

解决界面动画、交互反馈、特效制作等场景的需求。良好的动画设计能提升用户体验，使界面更加生动自然。

## 动画库

### Framer Motion
- **定位**：React 动画库
- **特点**：声明式 API、手势支持、布局动画
- **适用场景**：React 项目交互动画

### GSAP
- **定位**：专业动画库，功能强大
- **特点**：性能优秀、时间线控制、插件生态
- **适用场景**：复杂动画、SVG 动画、滚动驱动动画

### Anime.js
- **定位**：轻量级动画库
- **特点**：简单易用、支持多种属性、链式调用
- **适用场景**：轻量级动画需求

### Motion One
- **定位**：现代动画库
- **特点**：WAAPI 封装、体积小、性能好
- **适用场景**：现代浏览器、追求性能

## 动效设计

### Lottie
- **定位**：AE 动画导出，矢量动画
- **特点**：设计工具导出、JSON 格式、跨平台
- **适用场景**：品牌动画、图标动画、引导动画

### Spritesmith
- **定位**：雪碧图动画
- **特点**：帧动画、兼容性好
- **适用场景**：游戏、角色动画

### CSS Animations
- **定位**：原生 CSS 动画
- **特点**：无需 JS、性能好、简单场景
- **适用场景**：简单过渡、悬停效果

## 交互增强

### Interact.js
- **定位**：拖拽交互库
- **特点**：拖拽、缩放、旋转、多点触控
- **适用场景**：图片编辑、可拖拽元素

### Draggable
- **定位**：拖拽排序
- **特点**：平滑动画、滚动支持、触摸设备
- **适用场景**：列表排序、看板

### Sortable.js
- **定位**：列表排序
- **特点**：轻量、无依赖、支持多列表
- **适用场景**：拖拽排序、列表管理

## 滚动动画

### Scroll-driven Animations
- **定位**：原生 CSS 滚动动画
- **特点**：无需 JS、性能最佳
- **适用场景**：视差效果、滚动进度

### GSAP ScrollTrigger
- **定位**：滚动驱动动画插件
- **特点**：精确控制、回调丰富、支持钉住
- **适用场景**：滚动叙事、产品展示

### AOS (Animate On Scroll)
- **定位**：滚动触发动画
- **特点**：简单易用、预设动画
- **适用场景**：落地页、简单滚动动画

## 页面过渡

### View Transitions API
- **定位**：原生页面过渡
- **特点**：浏览器原生、共享元素过渡
- **适用场景**：SPA 路由过渡、元素变换

### React Transition Group
- **定位**：React 过渡动画
- **特点**：进入/退出动画、列表过渡
- **适用场景**：React 组件过渡

## 性能优化

### 使用 CSS transform
```css
/* 推荐 - 触发 GPU 加速 */
.element {
  transform: translateX(100px);
}

/* 避免 - 触发重排 */
.element {
  left: 100px;
}
```

### will-change 提示
```css
.element {
  will-change: transform, opacity;
}
```

### 使用 requestAnimationFrame
- 确保 60fps 流畅度
- 自动暂停处理
- 与浏览器刷新同步

## 动画原则

### 用户体验
- **目的性**：动画应服务于功能
- **适度**：不过度使用动画
- **可关闭**：尊重 prefers-reduced-motion

### 性能优先
- 使用 transform 和 opacity
- 避免重排重绘属性
- 合理使用 will-change

### 自然流畅
- 遵循物理规律
- 使用合适的缓动函数
- 保持动画一致性

## 技术选型建议

| 场景 | 推荐方案 | 理由 |
|------|---------|------|
| React 交互动画 | Framer Motion | 声明式、API 友好 |
| 复杂动画 | GSAP | 功能强大、生态完善 |
| 简单过渡 | CSS Transitions | 无依赖、性能好 |
| 拖拽排序 | Sortable.js | 轻量、功能完整 |
| AE 动画 | Lottie | 设计工具对接 |
| 滚动叙事 | GSAP ScrollTrigger | 控制精确 |
| 品牌动画 | Lottie + GSAP | 高质量输出 |

## 常见问题

### 动画卡顿
- 使用 GPU 加速属性
- 减少 JS 计算
- 使用 will-change
- 避免复杂选择器

### 移动端性能
- 减少动画数量
- 使用简化版本
- 利用硬件加速
- 适当降级

### 无障碍访问
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
