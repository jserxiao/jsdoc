## 技术选型指南

| 业务场景 | 推荐方案 | 核心技术 |
|---------|---------|---------|
| 企业后台管理系统 | React + Ant Design | React, Ant Design, Redux Toolkit, React Query |
| 移动端 H5 应用 | Vue + Vant | Vue 3, Vant, Pinia, rem/vw 适配 |
| 跨平台 App | React Native / Flutter | React Native 或 Flutter 全家桶 |
| 小程序多端 | uni-app / Taro | Vue 语法 + 多端编译 |
| 数据大屏 | Vue + ECharts | Vue 3, ECharts, DataV, Three.js |
| 微前端架构 | qiankun + Vue/React | qiankun, Module Federation |
| 组件库开发 | Vite + TypeScript | Vite, TypeScript, Storybook |
| 静态官网 | Astro | Astro, Markdown, 静态生成 |
| 全栈应用 | Next.js | Next.js, Prisma, PostgreSQL |
| 桌面应用 | Electron / Tauri | Electron 或 Tauri + React/Vue |

## 学习路径

1. **基础阶段**：掌握 JavaScript、TypeScript、HTML、CSS 核心语言
2. **应用开发**：选择一个框架深入学习，完成完整项目
3. **场景深入**：根据业务场景学习移动端、可视化、测试等专项技能
4. **工程化阶段**：掌握构建工具、代码规范、CI/CD
5. **架构进阶**：微前端、Monorepo、性能优化、系统设计
6. **全栈拓展**：Node.js 服务端、数据库、部署运维
