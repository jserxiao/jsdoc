# 性能优化与监控

## 概述

性能优化与监控是保障用户体验和系统稳定性的关键环节。本模块从加载优化、渲染优化、资源优化、网络优化和前端监控等多个维度，提供系统化的解决方案。

## 性能指标体系

### Web Vitals 核心指标

| 指标 | 名称 | 优秀值 | 需改进 | 差 | 说明 |
|------|------|--------|--------|-----|------|
| LCP | 最大内容绘制 | ≤ 2.5s | 2.5-4s | > 4s | 页面主要内容加载完成时间 |
| INP | 交互到下一次绘制 | ≤ 200ms | 200-500ms | > 500ms | 用户交互响应速度 |
| CLS | 累积布局偏移 | ≤ 0.1 | 0.1-0.25 | > 0.25 | 页面视觉稳定性 |
| FCP | 首次内容绘制 | ≤ 1.8s | 1.8-3s | > 3s | 首次渲染时间 |
| TTFB | 首字节时间 | ≤ 800ms | 800-1800ms | > 1800ms | 服务器响应速度 |
| FID | 首次输入延迟 | ≤ 100ms | 100-300ms | > 300ms | 首次交互响应 |

### 性能指标采集方式

```javascript
// 使用 Performance API 采集
const timing = performance.getEntriesByType('navigation')[0];

// 页面加载时间线
const metrics = {
    // DNS 查询
    dns: timing.domainLookupEnd - timing.domainLookupStart,
    // TCP 连接
    tcp: timing.connectEnd - timing.connectStart,
    // SSL 握手
    ssl: timing.secureConnectionStart > 0 
        ? timing.connectEnd - timing.secureConnectionStart : 0,
    // TTFB
    ttfb: timing.responseStart - timing.requestStart,
    // 内容下载
    download: timing.responseEnd - timing.responseStart,
    // DOM 解析
    domParse: timing.domInteractive - timing.responseEnd,
    // DOM Ready
    domReady: timing.domContentLoadedEventEnd - timing.startTime,
    // 页面完全加载
    load: timing.loadEventEnd - timing.startTime
};
```

## 优化策略清单

### 加载优化

- [ ] **代码分割** - 路由懒加载、组件懒加载
- [ ] **资源压缩** - JS/CSS/图片压缩
- [ ] **预加载** - preload 关键资源、prefetch 未来资源
- [ ] **CDN 部署** - 静态资源就近访问
- [ ] **传输压缩** - Gzip/Brotli
- [ ] **Service Worker** - 离线缓存、资源预缓存

### 渲染优化

- [ ] **虚拟列表** - 大数据列表按需渲染
- [ ] **防抖节流** - 高频事件控制
- [ ] **减少重排** - 批量 DOM 操作
- [ ] **CSS 优化** - 避免 @import、简化选择器
- [ ] **Web Worker** - 耗时计算分离线程
- [ ] **requestAnimationFrame** - 动画优化

### 资源优化

- [ ] **图片格式** - WebP/AVIF 优先
- [ ] **响应式图片** - srcset、picture 元素
- [ ] **字体优化** - WOFF2、字体子集化
- [ ] **Tree Shaking** - 移除无用代码
- [ ] **代码分割** - 按路由/功能拆分
- [ ] **缓存策略** - 强缓存 + 协商缓存

### 网络优化

- [ ] **HTTP/2** - 多路复用、头部压缩
- [ ] **域名分片** - 并行请求（HTTP/1.1）
- [ ] **预连接** - preconnect 提前建立连接
- [ ] **请求合并** - 减少请求数
- [ ] **请求缓存** - 避免重复请求

### 监控建设

- [ ] **错误监控** - JS 错误、Promise 错误、资源错误
- [ ] **性能监控** - Web Vitals 采集
- [ ] **行为监控** - PV/UV、点击、停留时长
- [ ] **请求监控** - 慢请求、错误请求
- [ ] **告警机制** - 关键指标告警
