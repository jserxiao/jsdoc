# 性能优化与监控

## 概述

解决性能瓶颈、用户体验优化、线上监控等场景的需求。性能优化是提升用户体验的关键，也是前端工程师的核心能力之一。

## 性能分析工具

### Lighthouse
- **定位**：性能审计工具
- **特点**：综合评分、优化建议、多维度分析
- **适用场景**：全面性能评估、优化前基准

### Chrome DevTools Performance
- **定位**：性能分析
- **特点**：帧率分析、CPU 占用、内存快照
- **适用场景**：细粒度性能问题定位

### Webpack Bundle Analyzer
- **定位**：包体积分析
- **特点**：可视化依赖树、找出大模块
- **适用场景**：打包优化、减少体积

### Rollup Visualizer
- **定位**：打包产物可视化
- **特点**：模块依赖图、Tree Shaking 分析
- **适用场景**：库开发、构建产物分析

## 加载性能优化

### 路由懒加载
- **原理**：代码分割，按需加载
- **实现**：动态 import、React.lazy、Vue 异步组件
- **效果**：减少首屏加载体积

### 图片懒加载
- **原理**：延迟加载可视区域外的图片
- **实现**：Intersection Observer、loading="lazy"
- **效果**：减少首屏请求数

### 预加载与预获取
- **preload**：关键资源提前加载
- **prefetch**：未来可能需要的资源空闲时加载
- **preconnect**：提前建立连接
- **效果**：缩短资源加载时间

### Service Worker
- **定位**：离线缓存
- **特点**：拦截请求、离线可用、后台同步
- **适用场景**：PWA、离线应用

## 渲染性能优化

### 虚拟列表
- **问题**：大数据列表渲染卡顿
- **方案**：只渲染可视区域 + 缓冲区
- **实现**：react-window、vue-virtual-scroller
- **效果**：万级数据流畅滚动

### 防抖与节流
- **防抖**：事件停止触发后执行
- **节流**：固定频率执行
- **适用场景**：搜索输入、滚动事件、窗口 resize
- **效果**：减少高频操作性能消耗

### requestAnimationFrame
- **定位**：动画性能优化
- **特点**：与浏览器刷新率同步、自动暂停
- **适用场景**：动画、Canvas 绘制
- **效果**：流畅动画、避免掉帧

### Web Worker
- **定位**：耗时计算分离线程
- **特点**：后台线程执行、不阻塞主线程
- **适用场景**：大数据计算、加密解密
- **效果**：保持 UI 响应

## 资源优化

### 图片优化
- **格式选择**：WebP/AVIF 优先，兼容降级
- **压缩**：TinyPNG、ImageOptim
- **响应式图片**：srcset、picture 元素
- **CDN 加速**：图片 CDN、自适应压缩

### 代码优化
- **Tree Shaking**：移除无用代码
- **代码分割**：按路由/功能拆分
- **压缩混淆**：Terser、cssnano
- **Gzip/Brotli**：传输压缩

### 缓存策略
- **强缓存**：Cache-Control、Expires
- **协商缓存**：ETag、Last-Modified
- **Service Worker 缓存**
- **CDN 缓存**

## 监控与埋点

### Sentry
- **定位**：错误监控平台
- **特点**：实时报警、堆栈追踪、SourceMap
- **适用场景**：线上错误监控、问题定位

### Google Analytics
- **定位**：用户行为分析
- **特点**：流量分析、用户画像、转化追踪
- **适用场景**：产品数据分析

### 神策数据
- **定位**：用户行为分析
- **特点**：私有化部署、实时分析、埋点管理
- **适用场景**：企业级数据平台

### Web Vitals
- **定位**：核心指标采集
- **指标**：
  - LCP (Largest Contentful Paint) - 最大内容绘制
  - FID (First Input Delay) - 首次输入延迟
  - CLS (Cumulative Layout Shift) - 累积布局偏移
  - INP (Interaction to Next Paint) - 交互到下一次绘制
- **适用场景**：性能指标标准化监控

## 性能指标

### 加载指标
| 指标 | 说明 | 优秀值 |
|------|------|-------|
| FCP | 首次内容绘制 | < 1.8s |
| LCP | 最大内容绘制 | < 2.5s |
| TTFB | 首字节时间 | < 800ms |

### 交互指标
| 指标 | 说明 | 优秀值 |
|------|------|-------|
| FID | 首次输入延迟 | < 100ms |
| INP | 交互到下一次绘制 | < 200ms |
| TTI | 可交互时间 | < 3.8s |

### 视觉指标
| 指标 | 说明 | 优秀值 |
|------|------|-------|
| CLS | 累积布局偏移 | < 0.1 |

## 技术选型建议

| 场景 | 推荐方案 | 理由 |
|------|---------|------|
| 性能评估 | Lighthouse | 全面、标准化 |
| 错误监控 | Sentry | 功能完善、生态好 |
| 用户分析 | GA/神策 | 按需求选择 |
| 包体积分析 | Bundle Analyzer | 可视化、直观 |
| 虚拟列表 | react-window | 高性能、API 简单 |
