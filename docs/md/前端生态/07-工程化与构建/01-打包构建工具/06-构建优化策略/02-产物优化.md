# 产物优化

## 优化策略总览

| 策略 | 说明 | 效果 |
|------|------|------|
| 代码分割 | splitChunks、动态 import | 减少首屏加载 |
| Tree Shaking | 移除无用代码 | 减小体积 |
| Scope Hoisting | 作用域提升 | 减少闭包 |
| 压缩 | Terser、cssnano | 减小体积 |
| 资源压缩 | Gzip、Brotli | 传输优化 |

## 代码分割

### Webpack splitChunks

```javascript
module.exports = {
  optimization: {
    splitChunks: {
      chunks: 'all',
      minSize: 20000,
      maxSize: 244000,
      minChunks: 1,
      maxAsyncRequests: 30,
      maxInitialRequests: 30,
      automaticNameDelimiter: '~',
      cacheGroups: {
        vendors: {
          test: /[\\/]node_modules[\\/]/,
          priority: -10,
          name: 'vendors'
        },
        default: {
          minChunks: 2,
          priority: -20,
          reuseExistingChunk: true
        }
      }
    }
  }
}
```

### 动态 import

```javascript
// 懒加载组件
const Home = () => import('./views/Home.vue')
const About = () => import('./views/About.vue')

// React.lazy
const Home = React.lazy(() => import('./views/Home'))

// Vue Router
const routes = [
  { path: '/', component: () => import('./views/Home.vue') }
]
```

### Vite manualChunks

```javascript
export default defineConfig({
  build: {
    rollupOptions: {
      output: {
        manualChunks: {
          vendor: ['vue', 'vue-router', 'pinia'],
          ui: ['element-plus'],
          utils: ['lodash-es', 'dayjs']
        }
      }
    }
  }
})
```

### 函数式 manualChunks

```javascript
manualChunks(id) {
  if (id.includes('node_modules')) {
    if (id.includes('vue') || id.includes('pinia')) {
      return 'vue-vendor'
    }
    if (id.includes('element-plus')) {
      return 'ui-vendor'
    }
    return 'vendor'
  }
}
```

## Tree Shaking

### 确保 ES Module 格式

```json
// package.json
{
  "sideEffects": false,
  "type": "module"
}
```

### 按需导入

```javascript
// ❌ 不推荐：全量导入
import _ from 'lodash'
import * as ElementPlus from 'element-plus'

// ✅ 推荐：按需导入
import debounce from 'lodash/debounce'
import { Button, Input } from 'element-plus'

// ✅ 使用 lodash-es
import { debounce } from 'lodash-es'
```

### Webpack 配置

```javascript
module.exports = {
  mode: 'production',  // 自动启用 Tree Shaking
  optimization: {
    usedExports: true,
    sideEffects: true
  }
}
```

## 压缩

### Terser (JavaScript)

```javascript
const TerserPlugin = require('terser-webpack-plugin')

module.exports = {
  optimization: {
    minimize: true,
    minimizer: [
      new TerserPlugin({
        parallel: true,
        terserOptions: {
          compress: {
            drop_console: true,
            drop_debugger: true,
            pure_funcs: ['console.log']
          },
          format: {
            comments: false
          }
        },
        extractComments: false
      })
    ]
  }
}
```

### cssnano (CSS)

```javascript
const CssMinimizerPlugin = require('css-minimizer-webpack-plugin')

module.exports = {
  optimization: {
    minimizer: [
      new CssMinimizerPlugin({
        minimizerOptions: {
          preset: ['default', {
            discardComments: { removeAll: true }
          }]
        }
      })
    ]
  }
}
```

### Vite 压缩

```javascript
export default defineConfig({
  build: {
    minify: 'esbuild',  // 'esbuild' | 'terser'
    terserOptions: {
      compress: {
        drop_console: true
      }
    }
  }
})
```

## Gzip/Brotli 压缩

### Webpack Gzip

```javascript
const CompressionPlugin = require('compression-webpack-plugin')

module.exports = {
  plugins: [
    new CompressionPlugin({
      algorithm: 'gzip',
      test: /\.(js|css|html|svg)$/,
      threshold: 10240,
      minRatio: 0.8
    })
  ]
}
```

### Brotli 压缩

```javascript
new CompressionPlugin({
  algorithm: 'brotliCompress',
  test: /\.(js|css|html|svg)$/,
  threshold: 10240,
  minRatio: 0.8
})
```

### Vite 压缩插件

```javascript
import compression from 'vite-plugin-compression'

export default defineConfig({
  plugins: [
    compression({
      algorithm: 'gzip'
    }),
    compression({
      algorithm: 'brotliCompress',
      ext: '.br'
    })
  ]
})
```

## 字体优化

### 字体子集化

```javascript
// 只保留需要的字符
import fontmin from 'fontmin'

new fontmin()
  .src('fonts/*.ttf')
  .use(fontmin.glyph({
    text: '需要的字符',
    hinting: false
  }))
  .dest('build/fonts')
  .run()
```

### 字体预加载

```html
<link rel="preload" href="/fonts/font.woff2" as="font" type="font/woff2" crossorigin>
```

## 移除无用代码

### 移除 console

```javascript
// Vite
export default defineConfig({
  esbuild: {
    drop: ['console', 'debugger']
  }
})

// Webpack (Terser)
{
  terserOptions: {
    compress: {
      drop_console: true,
      drop_debugger: true
    }
  }
}
```

### 移除 CSS 无用样式

```javascript
const PurgeCSSPlugin = require('purgecss-webpack-plugin')
const glob = require('glob')

module.exports = {
  plugins: [
    new PurgeCSSPlugin({
      paths: glob.sync(`${path.join(__dirname, 'src')}/**/*`, { nodir: true })
    })
  ]
}
```
