# 速度优化

## 优化策略总览

| 策略 | 说明 | 适用工具 | 效果 |
|------|------|----------|------|
| 持久化缓存 | 缓存编译结果 | Webpack、Vite | ⭐⭐⭐⭐⭐ |
| 并行构建 | 多线程/多进程编译 | thread-loader | ⭐⭐⭐⭐ |
| 缩小范围 | include/exclude | 所有工具 | ⭐⭐⭐⭐ |
| SWC/esbuild | 替代 Babel | Webpack、Vite | ⭐⭐⭐⭐⭐ |

## 持久化缓存

### Webpack 文件系统缓存

```javascript
// webpack.config.js
module.exports = {
  cache: {
    type: 'filesystem',
    cacheDirectory: 'node_modules/.cache/webpack',
    buildDependencies: {
      config: [__filename]
    }
  }
}
```

### Babel Loader 缓存

```javascript
{
  loader: 'babel-loader',
  options: {
    cacheDirectory: true,
    cacheCompression: false
  }
}
```

### Vite 缓存

```javascript
// vite.config.js
export default defineConfig({
  cacheDir: 'node_modules/.vite',
  
  optimizeDeps: {
    // 强制预构建
    include: ['vue', 'vue-router', 'pinia']
  }
})
```

## 并行构建

### thread-loader (Webpack)

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        use: [
          {
            loader: 'thread-loader',
            options: {
              workers: require('os').cpus().length - 1,
              poolTimeout: 2000
            }
          },
          'babel-loader'
        ]
      }
    ]
  }
}
```

### HappyPack (Webpack 4)

```javascript
const HappyPack = require('happypack')

module.exports = {
  plugins: [
    new HappyPack({
      id: 'js',
      threads: 4,
      loaders: ['babel-loader']
    })
  ]
}
```

## 使用 SWC 替代 Babel

### Webpack + swc-loader

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        use: {
          loader: 'swc-loader',
          options: {
            jsc: {
              parser: {
                syntax: 'ecmascript',
                jsx: true
              },
              transform: {
                react: {
                  runtime: 'automatic'
                }
              },
              target: 'es2015'
            }
          }
        }
      }
    ]
  }
}
```

### Vite + @vitejs/plugin-react-swc

```javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react-swc'

export default defineConfig({
  plugins: [react()]
})
```

## 使用 esbuild

### Webpack + esbuild-loader

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        use: {
          loader: 'esbuild-loader',
          options: {
            loader: 'js',
            target: 'es2015'
          }
        }
      },
      {
        test: /\.tsx?$/,
        use: {
          loader: 'esbuild-loader',
          options: {
            loader: 'tsx',
            target: 'es2015'
          }
        }
      }
    ]
  }
}
```

## 缩小编译范围

### include/exclude

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        include: path.resolve(__dirname, 'src'),
        exclude: /node_modules/,
        use: 'babel-loader'
      }
    ]
  }
}
```

### resolve 配置

```javascript
module.exports = {
  resolve: {
    // 明确扩展名，减少尝试次数
    extensions: ['.js', '.ts', '.json'],
    
    // 路径别名
    alias: {
      '@': path.resolve(__dirname, 'src')
    },
    
    // 限制搜索目录
    modules: ['node_modules']
  }
}
```

## noParse 优化

```javascript
module.exports = {
  module: {
    // 不解析已知无依赖的大型库
    noParse: /jquery|lodash/
  }
}
```

## DLL 预编译 (Webpack)

```javascript
// webpack.dll.js
const path = require('path')
const webpack = require('webpack')

module.exports = {
  entry: {
    vendor: ['vue', 'vue-router', 'pinia']
  },
  output: {
    path: path.join(__dirname, 'dll'),
    filename: '[name].js',
    library: '[name]'
  },
  plugins: [
    new webpack.DllPlugin({
      path: path.join(__dirname, 'dll', '[name]-manifest.json'),
      name: '[name]'
    })
  ]
}

// webpack.config.js
const webpack = require('webpack')

module.exports = {
  plugins: [
    new webpack.DllReferencePlugin({
      manifest: require('./dll/vendor-manifest.json')
    })
  ]
}
```

## SourceMap 优化

```javascript
module.exports = {
  // 开发环境
  devtool: 'eval-cheap-module-source-map',
  
  // 生产环境
  devtool: 'source-map',
  
  // 不需要 SourceMap
  devtool: false
}
```

### SourceMap 类型对比

| 类型 | 构建速度 | 重建速度 | 质量 |
|------|----------|----------|------|
| `eval` | 最快 | 最快 | 无映射 |
| `eval-cheap-source-map` | 快 | 快 | 转换后代码 |
| `eval-cheap-module-source-map` | 中 | 快 | 源代码 |
| `source-map` | 慢 | 慢 | 源代码 |
