# 核心特性

## 极致速度

esbuild 用 Go 语言编写，充分利用多核 CPU。

### 性能对比

| 项目 | Webpack | esbuild | 提升 |
|------|---------|---------|------|
| three.js | 37.69s | 0.33s | 114x |
| lodash | 3.26s | 0.16s | 20x |
| d3 | 7.66s | 0.24s | 32x |

### 速度来源

1. **Go 语言**：编译为原生代码，无解释开销
2. **并行处理**：充分利用多核 CPU
3. **内存高效**：从内存到内存，减少数据复制
4. **增量编译**：只重新编译变化的部分

## 内置功能

### TypeScript 支持

```typescript
// 无需配置，直接编译
interface User {
  name: string
  age: number
}

const user: User = { name: 'John', age: 30 }
```

```bash
npx esbuild src/index.ts --bundle --outfile=dist/bundle.js
```

### JSX 支持

```javascript
// React JSX
function App() {
  return <div>Hello World</div>
}

// 配置 JSX 工厂
npx esbuild src/index.jsx --bundle \
  --loader:.jsx=jsx \
  --jsx-factory=h \
  --jsx-fragment=Fragment
```

### CSS 支持

```css
/* style.css */
.container {
  color: red;
}
```

```javascript
// 自动处理 CSS
import './style.css'
```

### JSON 支持

```javascript
// 直接导入 JSON
import pkg from './package.json'
console.log(pkg.version)
```

## 零配置使用

```bash
# 基础打包
npx esbuild src/index.js --bundle --outfile=dist/bundle.js

# 压缩
npx esbuild src/index.js --bundle --minify --outfile=dist/bundle.min.js

# 监听模式
npx esbuild src/index.js --bundle --watch --outfile=dist/bundle.js

# 开发服务器
npx esbuild src/index.js --bundle --servedir=.
```

## 目标环境

```javascript
const esbuild = require('esbuild')

esbuild.build({
  entryPoints: ['src/index.ts'],
  bundle: true,
  outfile: 'dist/bundle.js',
  target: [
    'es2020',
    'chrome80',
    'firefox75',
    'safari13',
    'edge79'
  ]
})
```

## Tree Shaking

```javascript
// utils.js
export function used() { return 'used' }
export function unused() { return 'unused' }

// main.js
import { used } from './utils'
console.log(used())

// 输出：unused 被移除
```

## 代码分割

```javascript
esbuild.build({
  entryPoints: {
    main: 'src/main.js',
    admin: 'src/admin.js'
  },
  bundle: true,
  outdir: 'dist',
  splitting: true,
  format: 'esm'
})
```

## 外部依赖

```javascript
esbuild.build({
  entryPoints: ['src/index.js'],
  bundle: true,
  outfile: 'dist/bundle.js',
  external: [
    'react',
    'react-dom',
    /^lodash/
  ]
})
```

## 定义变量

```javascript
esbuild.build({
  define: {
    'process.env.NODE_ENV': '"production"',
    'process.env.VERSION': '"1.0.0"'
  }
})
```

## Loader 类型

| Loader | 说明 |
|--------|------|
| `js` | JavaScript |
| `ts` | TypeScript |
| `tsx` | TypeScript + JSX |
| `jsx` | JavaScript + JSX |
| `css` | CSS |
| `json` | JSON |
| `text` | 文本文件 |
| `base64` | Base64 编码 |
| `dataurl` | Data URL |
| `file` | 文件路径 |
| `binary` | 二进制 |

```javascript
esbuild.build({
  loader: {
    '.png': 'dataurl',
    '.svg': 'file',
    '.txt': 'text'
  }
})
```
