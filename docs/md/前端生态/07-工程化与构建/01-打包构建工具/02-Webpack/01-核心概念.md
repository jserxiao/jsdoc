# 核心概念

Webpack 有五个核心概念：Entry、Output、Loader、Plugin、Mode。

## Entry（入口）

入口指示 Webpack 从哪个文件开始构建依赖图。

```javascript
// 单入口
module.exports = {
  entry: './src/index.js'
}

// 多入口
module.exports = {
  entry: {
    main: './src/index.js',
    admin: './src/admin.js'
  }
}

// 动态入口
module.exports = {
  entry: () => './src/index.js'
}
```

## Output（出口）

出口告诉 Webpack 在哪里输出构建产物。

```javascript
const path = require('path')

module.exports = {
  output: {
    // 输出目录
    path: path.resolve(__dirname, 'dist'),
    
    // 文件名
    filename: 'bundle.js',
    
    // 多入口时使用占位符
    filename: '[name].[contenthash].js',
    
    // 公共路径
    publicPath: '/assets/',
    
    // 静态资源文件名
    assetModuleFilename: 'images/[hash][ext]',
    
    // 清空输出目录
    clean: true
  }
}
```

### 文件名占位符

| 占位符 | 说明 |
|--------|------|
| `[name]` | 入口名称 |
| `[id]` | chunk id |
| `[hash]` | 构建哈希 |
| `[chunkhash]` | chunk 哈希 |
| `[contenthash]` | 内容哈希 |

## Loader（加载器）

Loader 让 Webpack 能够处理非 JavaScript 文件。

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [
          'style-loader',  // 将 CSS 注入 DOM
          'css-loader'     // 解析 CSS 文件
        ]
      },
      {
        test: /\.ts$/,
        use: 'ts-loader',
        exclude: /node_modules/
      }
    ]
  }
}
```

### Loader 执行顺序

Loader 从右到左、从下到上执行：

```javascript
use: ['style-loader', 'css-loader', 'sass-loader']
// 执行顺序：sass-loader → css-loader → style-loader
```

### Loader 配置选项

```javascript
module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        use: [
          {
            loader: 'babel-loader',
            options: {
              presets: ['@babel/preset-env'],
              cacheDirectory: true
            }
          }
        ],
        include: path.resolve(__dirname, 'src'),
        exclude: /node_modules/
      }
    ]
  }
}
```

## Plugin（插件）

Plugin 用于执行更广泛的任务，如打包优化、资源管理等。

```javascript
const HtmlWebpackPlugin = require('html-webpack-plugin')
const MiniCssExtractPlugin = require('mini-css-extract-plugin')

module.exports = {
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html',
      filename: 'index.html'
    }),
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css'
    })
  ]
}
```

### 常用 Plugin

| Plugin | 作用 |
|--------|------|
| HtmlWebpackPlugin | 生成 HTML 文件 |
| MiniCssExtractPlugin | 提取 CSS 到单独文件 |
| DefinePlugin | 定义环境变量 |
| CopyWebpackPlugin | 复制静态资源 |
| BundleAnalyzerPlugin | 分析构建产物 |

## Mode（模式）

```javascript
module.exports = {
  mode: 'development' // 'production' | 'none'
}
```

### 模式差异

| 特性 | development | production |
|------|-------------|------------|
| `process.env.NODE_ENV` | `'development'` | `'production'` |
| 代码压缩 | ❌ | ✅ |
| Source Map | 启用 | 可配置 |
| Tree Shaking | ❌ | ✅ |
| Scope Hoisting | ❌ | ✅ |

## Module（模块）

Webpack 支持多种模块语法：

```javascript
// ES Module
import { name } from './module'
export default function() {}

// CommonJS
const { name } = require('./module')
module.exports = {}

// AMD
define(['./module'], function(module) {})

// 动态导入
import('./module').then(module => {})
```

## Chunk（代码块）

Chunk 是 Webpack 打包过程中的代码块：

```
Entry Chunk（入口块）
    ├── Runtime Chunk（运行时）
    └── Normal Chunk（普通块）
```

### Chunk 分割

```javascript
module.exports = {
  optimization: {
    splitChunks: {
      chunks: 'all',
      minSize: 20000,
      maxSize: 244000,
      cacheGroups: {
        vendors: {
          test: /[\\/]node_modules[\\/]/,
          name: 'vendors',
          chunks: 'all'
        }
      }
    },
    runtimeChunk: 'single'
  }
}
```

## Resolve（解析）

```javascript
module.exports = {
  resolve: {
    // 文件扩展名
    extensions: ['.js', '.ts', '.json'],
    
    // 路径别名
    alias: {
      '@': path.resolve(__dirname, 'src'),
      '@components': path.resolve(__dirname, 'src/components')
    },
    
    // 模块搜索路径
    modules: ['node_modules', 'src']
  }
}
```

## 完整配置示例

```javascript
const path = require('path')
const HtmlWebpackPlugin = require('html-webpack-plugin')

module.exports = {
  // 入口
  entry: './src/index.js',
  
  // 出口
  output: {
    path: path.resolve(__dirname, 'dist'),
    filename: '[name].[contenthash].js',
    clean: true
  },
  
  // 模式
  mode: 'production',
  
  // 模块规则
  module: {
    rules: [
      {
        test: /\.js$/,
        use: 'babel-loader',
        exclude: /node_modules/
      },
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  },
  
  // 插件
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html'
    })
  ],
  
  // 解析
  resolve: {
    extensions: ['.js', '.json'],
    alias: {
      '@': path.resolve(__dirname, 'src')
    }
  },
  
  // 优化
  optimization: {
    splitChunks: {
      chunks: 'all'
    }
  }
}
```
