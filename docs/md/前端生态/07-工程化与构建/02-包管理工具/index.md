# 包管理工具

## 概述

JavaScript 生态中的包管理工具用于管理项目依赖，主流工具包括：
- **npm**：Node.js 默认包管理器，最广泛使用
- **yarn**：Facebook 开发，性能和安全性更好
- **pnpm**：磁盘空间高效，安装速度快

## 三大工具对比

| 特性 | npm | yarn | pnpm |
|------|-----|------|------|
| 安装速度 | 较快 | 快 | 最快 |
| 磁盘空间 | 一般 | 一般 | 极省 |
| 幽灵依赖 | 有 | 有 | 无 |
| node_modules | 扁平化 | 扁平化 | 符号链接 |
| 锁文件 | package-lock.json | yarn.lock | pnpm-lock.yaml |
| Monorepo | workspaces | workspaces | workspace |
| 离线安装 | 支持 | 更好 | 支持 |
| 并行安装 | 是 | 是 | 是 |

## 选择建议

- **npm**：Node.js 默认自带，适合小型项目和快速原型
- **yarn**：适合中大型项目，需要更好的稳定性和性能
- **pnpm**：适合 Monorepo 项目，磁盘空间敏感场景
