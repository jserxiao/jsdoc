# 自定义脚手架

## 使用 plop 生成代码

### 安装

```bash
npm install -D plop
```

### plopfile.js 配置

```javascript
// plopfile.js
module.exports = function(plop) {
  plop.setGenerator('component', {
    description: '创建组件',
    prompts: [
      {
        type: 'input',
        name: 'name',
        message: '组件名称'
      }
    ],
    actions: [
      {
        type: 'add',
        path: 'src/components/{{name}}/index.tsx',
        templateFile: 'templates/component.hbs'
      },
      {
        type: 'add',
        path: 'src/components/{{name}}/index.module.css',
        templateFile: 'templates/component-css.hbs'
      }
    ]
  })
}
```

### 模板文件

```handlebars
// templates/component.hbs
import React from 'react'
import styles from './index.module.css'

interface {{pascalCase name}}Props {
  children?: React.ReactNode
}

export const {{pascalCase name}}: React.FC<{{pascalCase name}}Props> = ({ children }) => {
  return (
    <div className={styles.container}>
      {children}
    </div>
  )
}
```

### 运行

```bash
npx plop
# 或添加到 package.json scripts
npm run generate
```

## 使用 hygen 生成代码

### 安装

```bash
npm install -D hygen
```

### 初始化

```bash
npx hygen init self
npx hygen generator new component
```

### 模板结构

```
_templates/
└── component/
    └── new/
        ├── index.tsx.ejs.t
        └── index.css.ejs.t
```

### 模板文件

```yaml
---
to: src/components/<%= name %>/index.tsx
---
import React from 'react'
import styles from './index.module.css'

interface <%= h.changeCase.pascal(name) %>Props {
  children?: React.ReactNode
}

export const <%= h.changeCase.pascal(name) %>: React.FC<<%= h.changeCase.pascal(name) %>Props> = ({ children }) => {
  return (
    <div className={styles.container}>
      {children}
    </div>
  )
}
```

### 运行

```bash
npx hygen component new MyComponent
```

## plop vs hygen

| 特性 | plop | hygen |
|------|------|-------|
| 模板引擎 | Handlebars | EJS |
| 交互提示 | 支持 | 不支持 |
| 学习曲线 | 简单 | 中等 |
| 速度 | 一般 | 快 |
| 灵活性 | 高 | 高 |
