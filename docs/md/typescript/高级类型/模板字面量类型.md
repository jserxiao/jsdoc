# 模板字面量类型

模板字面量类型允许在类型层面操作字符串，类似于 JavaScript 的模板字符串。

## 基本语法

### 字符串拼接

```typescript
type World = 'world';
type Greeting = `hello ${World}`;
// 'hello world'

type EventName<T extends string> = `on${T}`;
type ClickEvent = EventName<'Click'>;  // 'onClick'
```

### 类型插值

```typescript
type Email<T extends string> = `${T}@example.com`;
type UserEmail = Email<'alice'>;  // 'alice@example.com'
```

## 字符串操作

### 内置工具类型

```typescript
// 转大写
type Upper = Uppercase<'hello'>;  // 'HELLO'

// 转小写
type Lower = Lowercase<'HELLO'>;  // 'hello'

// 首字母大写
type Cap = Capitalize<'hello'>;   // 'Hello'

// 首字母小写
type Uncap = Uncapitalize<'Hello'>;  // 'hello'
```

### 自定义字符串操作

```typescript
// 添加前缀
type AddPrefix<T extends string, P extends string> = `${P}${T}`;
type Prefixed = AddPrefix<'name', 'user_'>;  // 'user_name'

// 添加后缀
type AddSuffix<T extends string, S extends string> = `${T}${S}`;
type Suffixed = AddSuffix<'user', '_id'>;  // 'user_id'

// 包裹字符串
type Wrap<T extends string, W extends string> = `${W}${T}${W}`;
type Wrapped = Wrap<'hello', '"'>;  // '"hello"'
```

## 联合类型组合

### 笛卡尔积

```typescript
type Color = 'red' | 'blue' | 'green';
type Size = 'small' | 'medium' | 'large';

type ColorSize = `${Color}-${Size}`;
// 'red-small' | 'red-medium' | 'red-large' 
// | 'blue-small' | 'blue-medium' | 'blue-large' 
// | 'green-small' | 'green-medium' | 'green-large'
```

### 方法名生成

```typescript
type Method = 'get' | 'set' | 'delete';
type Entity = 'User' | 'Product' | 'Order';

type ApiMethod = `${Method}${Entity}`;
// 'getUser' | 'setUser' | 'deleteUser' 
// | 'getProduct' | 'setProduct' | 'deleteProduct' 
// | 'getOrder' | 'setOrder' | 'deleteOrder'
```

### 事件处理器

```typescript
type Event = 'click' | 'focus' | 'blur';
type Handler = `on${Capitalize<Event>}`;

type Handlers = {
  [K in Handler]: () => void;
};
// {
//   onClick: () => void;
//   onFocus: () => void;
//   onBlur: () => void;
// }
```

## 字符串模式匹配

### infer 提取

```typescript
// 提取前缀
type RemovePrefix<T extends string, P extends string> = 
  T extends `${P}${infer Rest}` ? Rest : T;

type Result = RemovePrefix<'user_name', 'user_'>;  // 'name'

// 提取后缀
type RemoveSuffix<T extends string, S extends string> = 
  T extends `${infer Rest}${S}` ? Rest : T;

type Result2 = RemoveSuffix<'user_id', '_id'>;  // 'user'
```

### 字符串分割

```typescript
type Split<S extends string, D extends string> = 
  S extends `${infer Head}${D}${infer Tail}` 
    ? [Head, ...Split<Tail, D>] 
    : [S];

type Parts = Split<'a-b-c', '-'>;  // ['a', 'b', 'c']
```

### 替换字符串

```typescript
type Replace<S extends string, From extends string, To extends string> = 
  S extends `${infer Before}${From}${infer After}` 
    ? `${Before}${To}${After}` 
    : S;

type Result = Replace<'hello world', ' ', '-'>;  // 'hello-world'
```

### 驼峰转换

```typescript
type CamelCase<S extends string> = 
  S extends `${infer Head}-${infer Tail}` 
    ? `${Head}${Capitalize<CamelCase<Tail>>}` 
    : S;

type Result = CamelCase<'hello-world-example'>;  // 'helloWorldExample'
```

### 蛇形转换

```typescript
type SnakeCase<S extends string> = 
  S extends `${infer C}${infer Rest}` 
    ? C extends Uppercase<C> 
      ? `${Lowercase<C>}${SnakeCase<Rest>}` 
      : `_${Lowercase<C>}${SnakeCase<Rest>}` 
    : S;
```

## 实际应用

### 类型安全的 getter/setter

```typescript
type Getters<T> = {
  [K in keyof T as `get${Capitalize<K & string>}`]: () => T[K];
};

type Setters<T> = {
  [K in keyof T as `set${Capitalize<K & string>}`]: (value: T[K]) => void;
};

interface User {
  name: string;
  age: number;
}

type UserGetters = Getters<User>;
// { getName: () => string; getAge: () => number; }

type UserSetters = Setters<User>;
// { setName: (value: string) => void; setAge: (value: number) => void; }
```

### 类型安全的事件系统

```typescript
type Events = {
  click: { x: number; y: number };
  focus: { target: HTMLElement };
};

type EventHandler<T extends keyof Events> = `on${Capitalize<T>}`;

type EventHandlers = {
  [K in keyof Events as EventHandler<K>]: (event: Events[K]) => void;
};
```
