## moduleResolution

### 解析策略

```json
{
  "compilerOptions": {
    "moduleResolution": "node"
  }
}
```

| 策略 | 说明 |
|------|------|
| `classic` | TypeScript 旧版策略（已弃用） |
| `node` | Node.js 风格解析（Node10） |
| `node16` / `nodenext` | Node.js ESM 解析 |
| `bundler` | 打包工具解析（Vite、Webpack 5+） |

### classic 策略

```
// 导入: import { x } from './module'
// 从 /root/src/folder/A.ts 开始查找

/root/src/folder/module.ts
/root/src/module.ts
/root/module.ts
/module.ts
```

### node 策略

```
// 导入: import { x } from './module'
// 从 /root/src/folder/A.ts 开始查找

/root/src/folder/module.ts
/root/src/folder/module.tsx
/root/src/folder/module.d.ts
/root/src/folder/module/package.json (types 字段)
/root/src/folder/module/index.ts
/root/src/folder/module/index.tsx
/root/src/folder/module/index.d.ts
```

### bundler 策略

用于打包工具（Vite、Webpack 5+）：

```json
{
  "compilerOptions": {
    "moduleResolution": "bundler",
    "allowImportingTsExtensions": true,
    "resolveJsonModule": true
  }
}
```

## baseUrl 和 paths

### baseUrl

设置模块解析的根目录：

```json
{
  "compilerOptions": {
    "baseUrl": "."
  }
}
```

```typescript
// 从任意位置导入
import { Button } from 'src/components/Button';
```

### paths

配置路径别名：

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"],
      "@components/*": ["src/components/*"],
      "@utils/*": ["src/utils/*"],
      "@hooks/*": ["src/hooks/*"],
      "@types/*": ["src/types/*"]
    }
  }
}
```

```typescript
// 不使用别名
import { Button } from '../../../components/Button';
import { formatDate } from '../../../utils/date';

// 使用别名
import { Button } from '@components/Button';
import { formatDate } from '@utils/date';
```

### 相对路径和别名混用

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"],
      "@app/*": ["src/app/*"],
      "@shared/*": ["../shared/src/*"]
    }
  }
}
```

## 模块导入选项

### resolveJsonModule

允许导入 JSON 文件：

```json
{
  "compilerOptions": {
    "resolveJsonModule": true
  }
}
```

```typescript
import config from './config.json';
import { name } from './package.json';
```

### allowArbitraryExtensions

允许任意文件扩展名：

```json
{
  "compilerOptions": {
    "allowArbitraryExtensions": true
  }
}
```

```typescript
// 需要 data.svg.d.ts 声明文件
import svg from './data.svg';
```

### allowImportingTsExtensions

允许导入 `.ts` 扩展名：

```json
{
  "compilerOptions": {
    "allowImportingTsExtensions": true,
    "noEmit": true
  }
}
```

```typescript
import { Button } from './Button.ts';
```

### customConditions

自定义模块解析条件：

```json
{
  "compilerOptions": {
    "customConditions": ["my-condition"]
  }
}
```

## 类型解析

### typeRoots

指定类型声明文件的根目录：

```json
{
  "compilerOptions": {
    "typeRoots": [
      "./types",
      "./node_modules/@types"
    ]
  }
}
```

### types

显式指定要包含的类型声明：

```json
{
  "compilerOptions": {
    "types": ["node", "jest", "express"]
  }
}
```

```typescript
// 只包含指定的类型
import express from 'express';  // OK
import lodash from 'lodash';    // 如果 lodash 不在 types 中，不会有类型
```

### exclude type packages

```json
{
  "compilerOptions": {
    "types": []  // 不自动包含任何 @types 包
  }
}
```

## Node.js ESM 配置

```json
// Node.js ESM 项目
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "esModuleInterop": true
  }
}
```

```typescript
// 必须使用完整扩展名
import { foo } from './module.js';  // .js 而不是 .ts
```

## Vite/Webpack 项目配置

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "allowImportingTsExtensions": true,
    "resolveJsonModule": true,
    "isolatedModules": true,
    "noEmit": true,
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    }
  }
}
```
