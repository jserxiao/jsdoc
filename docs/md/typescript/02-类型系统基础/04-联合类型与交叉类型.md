# 联合类型与交叉类型

## 概述

联合类型和交叉类型是 TypeScript 中组合类型的两种主要方式。

## 一、联合类型 (Union Types)

### 1.1 基本概念

联合类型表示值可以是多种类型之一,使用 `|` 分隔:

```typescript
// 基本联合类型
let id: string | number

id = '123'   // ✅
id = 123     // ✅
// id = true  // ❌ 编译错误
```

### 1.2 使用场景

```typescript
// 函数参数
function formatValue(value: string | number): string {
  if (typeof value === 'string') {
    return value.toUpperCase()
  } else {
    return value.toFixed(2)
  }
}

// 联合类型数组
let mixed: (string | number)[] = [1, 'two', 3, 'four']

// 对象属性
interface User {
  id: string | number
  name: string
}
```

### 1.3 类型缩小

```typescript
function process(value: string | number) {
  // value: string | number

  if (typeof value === 'string') {
    // value: string (类型缩小)
    return value.toUpperCase()
  } else {
    // value: number (类型缩小)
    return value * 2
  }
}
```

### 1.4 可辨识联合

```typescript
interface User {
  type: 'user'
  name: string
}

interface Admin {
  type: 'admin'
  name: string
  permissions: string[]
}

type Person = User | Admin

function process(person: Person) {
  switch (person.type) {
    case 'user':
      console.log('User:', person.name)
      break
    case 'admin':
      console.log('Admin:', person.name, person.permissions)
      break
  }
}
```

## 二、交叉类型 (Intersection Types)

### 2.1 基本概念

交叉类型将多个类型合并为一个类型,使用 `&` 连接:

```typescript
interface Name {
  name: string
}

interface Age {
  age: number
}

type Person = Name & Age

const person: Person = {
  name: 'Alice',
  age: 25
}
```

### 2.2 合并对象类型

```typescript
interface BusinessContact {
  name: string
  email: string
}

interface PersonalContact {
  name: string
  phone: string
}

type Contact = BusinessContact & PersonalContact

const contact: Contact = {
  name: 'Alice',
  email: 'alice@example.com',
  phone: '123-456-7890'
}
```

### 2.3 合并冲突

```typescript
interface A {
  prop: string
}

interface B {
  prop: number
}

type C = A & B

// prop 类型为 never (string & number 不兼容)
// const c: C = { prop: 'hello' }  // 编译错误
```

## 三、联合 vs 交叉

| 特性 | 联合类型 `|` | 交叉类型 `&` |
|------|-------------|-------------|
| 含义 | 或 (OR) | 且 (AND) |
| 类型范围 | 扩大 | 缩小 |
| 属性 | 满足任一类型 | 满足所有类型 |
| 方法调用 | 只能调用共同方法 | 可以调用所有方法 |

## 四、实战示例

### 4.1 配置合并

```typescript
interface DefaultConfig {
  timeout: number
  retries: number
}

interface UserConfig {
  timeout?: number
  retries?: number
  custom?: string
}

function mergeConfig(defaults: DefaultConfig, user: UserConfig): DefaultConfig {
  return { ...defaults, ...user }
}
```

### 4.2 条件类型

```typescript
type NonNullable<T> = T extends null | undefined ? never : T

type Extract<T, U> = T extends U ? T : never

type Exclude<T, U> = T extends U ? never : T
```

## 五、总结

- **联合类型**: 值可以是多种类型之一
- **交叉类型**: 值必须满足所有类型
- **类型缩小**: 使用类型守卫缩小联合类型范围
