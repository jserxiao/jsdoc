# 装饰器基础

装饰器是一种特殊声明，可以附加到类声明、方法、属性或参数上，修改其行为。

## 启用装饰器

### tsconfig.json 配置

```json
{
  "compilerOptions": {
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true
  }
}
```

### 安装 reflect-metadata

```bash
npm install reflect-metadata
```

## 装饰器类型

| 装饰器类型 | 目标 |
|------------|------|
| 类装饰器 | 类 |
| 方法装饰器 | 方法 |
| 属性装饰器 | 属性 |
| 参数装饰器 | 参数 |
| 访问器装饰器 | getter/setter |

## 执行顺序

```typescript
function ClassDecorator() {
  return function(target: any) {
    console.log('ClassDecorator');
  };
}

function MethodDecorator() {
  return function(target: any, key: string, descriptor: PropertyDescriptor) {
    console.log('MethodDecorator');
  };
}

function PropertyDecorator() {
  return function(target: any, key: string) {
    console.log('PropertyDecorator');
  };
}

function ParameterDecorator() {
  return function(target: any, key: string, index: number) {
    console.log('ParameterDecorator');
  };
}

@ClassDecorator()
class Example {
  @PropertyDecorator()
  property: string;

  @MethodDecorator()
  method(@ParameterDecorator() param: string) {}
}

// 执行顺序:
// 1. PropertyDecorator
// 2. ParameterDecorator
// 3. MethodDecorator
// 4. ClassDecorator
```

## 新装饰器提案

TypeScript 5.0+ 支持新的装饰器提案：

```typescript
// 新装饰器语法
function logged(value: Function, context: ClassMethodDecoratorContext) {
  return function(this: any, ...args: any[]) {
    console.log(`Calling ${String(context.name)}`);
    return value.apply(this, args);
  };
}

class Calculator {
  @logged
  add(a: number, b: number) {
    return a + b;
  }
}
```
