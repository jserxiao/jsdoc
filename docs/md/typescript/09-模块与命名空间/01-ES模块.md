# ES模块

## 概述

ES 模块是现代 JavaScript 的标准模块系统。

## 一、导出

### 命名导出

```typescript
// user.ts
export interface User {
  id: number
  name: string
}

export function createUser(name: string): User {
  return { id: Date.now(), name }
}

export const DEFAULT_NAME = 'Anonymous'
```

### 默认导出

```typescript
// user.ts
export default class User {
  constructor(public name: string) {}
}

// 或
class User {
  constructor(public name: string) {}
}

export default User
```

### 统一导出

```typescript
// utils.ts
const PI = 3.14159
function log(msg: string) { console.log(msg) }
function error(msg: string) { console.error(msg) }

export { PI, log, error }
export { log as logger }
```

## 二、导入

### 命名导入

```typescript
import { User, createUser } from './user'
import { User as UserType } from './user'
```

### 默认导入

```typescript
import User from './user'
import User, { createUser } from './user'
```

### 批量导入

```typescript
import * as UserModule from './user'

const user = UserModule.createUser('Alice')
```

### 动态导入

```typescript
async function loadModule() {
  const module = await import('./user')
  const user = module.createUser('Alice')
}
```

## 三、重导出

```typescript
// index.ts
export { User, createUser } from './user'
export * from './utils'
export { default as UserService } from './service'
```

## 四、实战示例

```typescript
// types/index.ts
export interface User {
  id: number
  name: string
}

// services/user.ts
import { User } from '../types'

export class UserService {
  async getUser(id: number): Promise<User> {
    const response = await fetch(`/api/users/${id}`)
    return response.json()
  }
}

// index.ts
export * from './types'
export * from './services'
```

## 五、总结

- 命名导出: `export { ... }`
- 默认导出: `export default ...`
- 命名导入: `import { ... } from ...`
- 动态导入: `import(...)`
