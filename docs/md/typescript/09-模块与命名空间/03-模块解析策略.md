## 解析策略类型

```json
{
  "compilerOptions": {
    "moduleResolution": "node"
  }
}
```

| 策略 | 说明 |
|------|------|
| `classic` | TypeScript 旧版策略 |
| `node` | Node.js 风格解析 |
| `node16` / `nodenext` | Node.js ESM 解析 |
| `bundler` | 打包工具解析 |

---

## Classic 策略

### 相对路径导入

```
// 导入: import { x } from './module'
// 从 /root/src/folder/A.ts 开始查找

/root/src/folder/module.ts
/root/src/module.ts
/root/module.ts
/module.ts
```

### 非相对路径导入

```
// 导入: import { x } from 'module'
// 从 /root/src/folder/A.ts 开始查找

/root/src/folder/module.ts
/root/src/module.ts
/root/module.ts
/module.ts
```

## Node 策略

### 相对路径导入

```
// 导入: import { x } from './module'
// 从 /root/src/folder/A.ts 开始查找

/root/src/folder/module.ts
/root/src/folder/module.tsx
/root/src/folder/module.d.ts
/root/src/folder/module/package.json (types 字段)
/root/src/folder/module/index.ts
/root/src/folder/module/index.tsx
/root/src/folder/module/index.d.ts
```

### 非相对路径导入

```
// 导入: import { x } from 'module'
// 从 /root/src/folder/A.ts 开始查找

/root/src/folder/node_modules/module.ts
/root/src/folder/node_modules/module.tsx
/root/src/folder/node_modules/module.d.ts
/root/src/folder/node_modules/module/package.json
/root/src/folder/node_modules/@types/module.d.ts
/root/src/node_modules/...
/root/node_modules/...
/node_modules/...
```

## Node16/NodeNext 策略

用于 Node.js ESM 项目：

```json
{
  "compilerOptions": {
    "module": "NodeNext",
    "moduleResolution": "NodeNext"
  }
}
```

```typescript
// 必须使用完整扩展名
import { foo } from './module.js';  // .js 而不是 .ts

// package.json exports 支持
import { something } from 'my-lib/subpath';
```

---

## Bundler 策略

用于打包工具（Vite、Webpack 5+）：

```json
{
  "compilerOptions": {
    "moduleResolution": "bundler"
  }
}
```

特点：
- 允许省略扩展名
- 支持 `package.json` exports
- 允许导入 `.ts` 文件

```typescript
// 无需扩展名
import { foo } from './module';

// 可以导入 .ts
import { bar } from './utils.ts';
```

## 路径映射

### baseUrl

```json
{
  "compilerOptions": {
    "baseUrl": "."
  }
}
```

### paths

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"],
      "@components/*": ["src/components/*"]
    }
  }
}
```

```typescript
import { Button } from '@components/Button';
```
