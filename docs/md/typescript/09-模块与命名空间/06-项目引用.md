## 基本概念

项目引用的优势：

- 增量编译：只编译修改的部分
- 代码组织：清晰的依赖关系
- 跨项目类型检查
- 减少编译时间

## 配置项目引用

### monorepo 结构

```
project/
├── tsconfig.json          # 根配置
├── packages/
│   ├── shared/
│   │   ├── tsconfig.json
│   │   └── src/
│   ├── core/
│   │   ├── tsconfig.json
│   │   └── src/
│   └── app/
│       ├── tsconfig.json
│       └── src/
```

### 根配置

```json
// tsconfig.json
{
  "references": [
    { "path": "./packages/shared" },
    { "path": "./packages/core" },
    { "path": "./packages/app" }
  ],
  "files": []
}
```

### 子项目配置

```json
// packages/shared/tsconfig.json
{
  "compilerOptions": {
    "composite": true,
    "declaration": true,
    "declarationMap": true,
    "sourceMap": true,
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "include": ["src"]
}
```

```json
// packages/core/tsconfig.json
{
  "compilerOptions": {
    "composite": true,
    "declaration": true,
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "references": [
    { "path": "../shared" }
  ],
  "include": ["src"]
}
```

```json
// packages/app/tsconfig.json
{
  "compilerOptions": {
    "composite": true,
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "references": [
    { "path": "../shared" },
    { "path": "../core" }
  ],
  "include": ["src"]
}
```

## 增量编译

### composite 选项

```json
{
  "compilerOptions": {
    "composite": true
  }
}
```

`composite: true` 会自动启用：

- `declaration`: 生成声明文件
- `strict`: 严格模式
- `declarationMap`: 声明文件映射

### 增量编译命令

```bash
# 构建所有项目
tsc --build

# 构建特定项目
tsc --build packages/shared

# 强制重建
tsc --build --force

# 清理输出
tsc --build --clean
```

### .d.ts 输出

项目引用使用 `.d.ts` 文件进行类型检查，而不是源文件：

```
packages/shared/
├── src/
│   └── index.ts
└── dist/
    ├── index.js
    ├── index.d.ts      # 被其他项目使用
    └── index.d.ts.map
```

## 路径映射

### 在 monorepo 中配置路径别名

```json
// packages/app/tsconfig.json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@shared/*": ["../shared/src/*"],
      "@core/*": ["../core/src/*"]
    }
  },
  "references": [
    { "path": "../shared" },
    { "path": "../core" }
  ]
}
```

```typescript
// 使用路径别名
import { something } from '@shared/utils';
```

## 最佳实践

### 依赖方向

```
app → core → shared
```

- `shared`: 基础工具、类型
- `core`: 业务逻辑
- `app`: 应用入口

### 避免循环引用

```typescript
// ❌ 错误：循环引用
// app → core → app

// ✅ 正确：单向依赖
// app → core → shared
```

### 使用 TypeScript Project References

```bash
# 安装 TypeScript
npm install -D typescript

# 构建
npx tsc --build
```
