# 量词与分组

量词和分组是正则表达式中两个强大的特性。量词用于控制匹配的次数，分组用于组织模式和提取匹配内容。

## 量词

量词用于指定前面的元素可以出现多少次。

### 基本量词

```javascript
// *：0 次或多次（等价于 {0,}）
/a*/.test('');      // true
/a*/.test('a');     // true
/a*/.test('aaa');   // true
'aabbaa'.match(/a*/g); // ['aa', '', 'aa', '']

// +：1 次或多次（等价于 {1,}）
/a+/.test('');      // false
/a+/.test('a');     // true
/a+/.test('aaa');   // true
'aaa'.match(/a+/g); // ['aaa']

// ?：0 次或 1 次（等价于 {0,1}）
/a?/.test('');      // true
/a?/.test('a');     // true
/a?/.test('aa');    // true（只匹配一个 a）
'aaa'.match(/a?/g); // ['a', 'a', 'a', '']
```

### 花括号量词

```javascript
// {n}：恰好 n 次
/a{3}/.test('aa');    // false
/a{3}/.test('aaa');   // true
/a{3}/.test('aaaa');  // true（匹配前 3 个 a）
'aaa'.match(/a{2}/g); // ['aa']

// {n,}：至少 n 次
/a{2,}/.test('a');     // false
/a{2,}/.test('aa');    // true
/a{2,}/.test('aaaaa'); // true
'aaaa'.match(/a{2,}/g); // ['aaaa']

// {n,m}：n 到 m 次
/a{2,4}/.test('a');     // false
/a{2,4}/.test('aa');    // true
/a{2,4}/.test('aaaaa'); // true（只匹配 4 个 a）
'aaaaa'.match(/a{2,4}/g); // ['aaaa']

// 实际应用
const phoneNumber = /\d{3}-\d{4}-\d{4}/;
phoneNumber.test('010-1234-5678'); // true

const hexColor = /#[0-9a-fA-F]{6}/;
hexColor.test('#ff0000'); // true

const password = /[a-zA-Z0-9]{8,16}/;
password.test('abc12345'); // true
```

### 量词作用于分组

```javascript
// 量词可以作用于分组
/(ab)+/.test('ab');     // true
/(ab)+/.test('abab');   // true
/(ab)+/.test('ababab'); // true
'ababab'.match(/(ab)+/); // ['ababab', 'ab']

// 量词作用于字符类
/[abc]+/.test('aabcc'); // true
'abcdef'.match(/[abc]+/g); // ['abc']
```

## 贪婪与非贪婪

### 贪婪匹配（默认）

贪婪量词会尽可能多地匹配字符。

```javascript
const str = '<div>content</div>';

// 贪婪匹配：尽可能多匹配
/<.+>/.exec(str);
// ['<div>content</div>']

// 原因：. 匹配到最后的 >
// 然后回溯到第一个 > 但整体匹配已经成功
```

### 非贪婪匹配（懒惰）

非贪婪量词会尽可能少地匹配字符。在量词后加 `?` 变为非贪婪。

```javascript
const str = '<div>content</div>';

// 非贪婪匹配：尽可能少匹配
/<.+?>/.exec(str);
// ['<div>']

// *?：0 次或多次，非贪婪
/a*?/.exec('aaa'); // ['']

// +?：1 次或多次，非贪婪
/a+?/.exec('aaa'); // ['a']

// ??：0 次或 1 次，非贪婪
/a??/.exec('a'); // ['']

// {n,}?：至少 n 次，非贪婪
/a{2,}?/.exec('aaaa'); // ['aa']

// {n,m}?：n 到 m 次，非贪婪
/a{2,4}?/.exec('aaaa'); // ['aa']
```

### 贪婪 vs 非贪婪对比

```javascript
const text = 'aaaa';

// 贪婪
/a+/.exec(text);   // ['aaaa']
/a{2,4}/.exec(text); // ['aaaa']

// 非贪婪
/a+?/.exec(text);  // ['a']
/a{2,4}?/.exec(text); // ['aa']

// 提取 HTML 标签
const html = '<p>text1</p><p>text2</p>';

// 贪婪：匹配整个字符串
/<p>.*<\/p>/.exec(html);
// ['<p>text1</p><p>text2</p>']

// 非贪婪：匹配每个标签对
/<p>.*?<\/p>/g.exec(html);
// ['<p>text1</p>']
html.match(/<p>.*?<\/p>/g);
// ['<p>text1</p>', '<p>text2</p>']
```

## 分组

分组使用括号 `()` 将多个字符组合成一个整体。

### 捕获分组

捕获分组会记住匹配的内容，可以在后续引用或提取。

```javascript
// 基本捕获
/(ab)+/.exec('ababab');
// ['ababab', 'ab']
// 结果[0]: 完整匹配
// 结果[1]: 第一个捕获分组

// 多个捕获分组
/(\d+)-(\d+)-(\d+)/.exec('2024-01-15');
// ['2024-01-15', '2024', '01', '15']

// 提取分组内容
const dateMatch = /(\d{4})-(\d{2})-(\d{2})/.exec('2024-01-15');
const [, year, month, day] = dateMatch;
console.log(year, month, day); // 2024 01 15

// match 方法
'2024-01-15'.match(/(\d{4})-(\d{2})-(\d{2})/);
// ['2024-01-15', '2024', '01', '15', index: 0, input: '2024-01-15']

// matchAll 方法（ES2020）
const text = '2024-01-15 and 2024-12-31';
const matches = [...text.matchAll(/(\d{4})-(\d{2})-(\d{2})/g)];
matches.forEach(m => {
  console.log(`Full: ${m[0]}, Year: ${m[1]}, Month: ${m[2]}, Day: ${m[3]}`);
});
```

### 非捕获分组

非捕获分组 `(?:...)` 只分组不捕获，不占用分组编号。

```javascript
// 捕获分组
/(ab)+/.exec('ababab');
// ['ababab', 'ab']

// 非捕获分组
/(?:ab)+/.exec('ababab');
// ['ababab']
// 注意：没有捕获 'ab'

// 混合使用
/(\d{4})-(?:\d{2})-(\d{2})/.exec('2024-01-15');
// ['2024-01-15', '2024', '15']
// 中间的月分没有被捕获

// 实际应用：匹配协议
const url = /^(?:https?|ftp):\/\/.+/;
url.test('http://example.com');  // true
url.test('https://example.com'); // true
url.test('ftp://example.com');   // true
```

### 命名分组（ES2018）

命名分组可以为捕获组指定名称，提高代码可读性。

```javascript
// 基本语法：(?<name>...)
const dateRegex = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const result = dateRegex.exec('2024-01-15');

console.log(result.groups);
// { year: '2024', month: '01', day: '15' }

// 解构使用
const { groups: { year, month, day } } = result;
console.log(year, month, day); // 2024 01 15

// 在 replace 中使用命名分组
'2024-01-15'.replace(
  /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/,
  '$<day>/$<month>/$<year>'
);
// '15/01/2024'

// 复杂示例
const emailRegex = /(?<username>[^@]+)@(?<domain>[^@]+)/;
const emailMatch = emailRegex.exec('user@example.com');
console.log(emailMatch.groups);
// { username: 'user', domain: 'example.com' }
```

## 反向引用

反向引用允许在正则表达式中引用之前捕获的内容。

### 数字引用

使用 `\1`、`\2` 等引用第 N 个捕获组。

```javascript
// 匹配重复字符
/(\w)\1/.test('aa');  // true
/(\w)\1/.test('ab');  // false
/(\w)\1/.test('bb');  // true

// 匹配 ABBA 模式
/(\w)(\w)\2\1/.test('abba'); // true
/(\w)(\w)\2\1/.test('baab'); // true
/(\w)(\w)\2\1/.test('abab'); // false

// 匹配 HTML 标签对
const tagPattern = /<(\w+)>.*?<\/\1>/;
tagPattern.test('<div>content</div>'); // true
tagPattern.test('<div>content</span>'); // false

// 匹配引号内的内容
const quotedString = /(['"])(.*?)\1/;
quotedString.exec('"hello"'); // ['"hello"', '"', 'hello']
quotedString.exec("'hello'"); // ["'hello'", "'", 'hello']

// 多个分组引用
/(\d)(\d)\1\2/.test('1212'); // true
/(\d)(\d)\1\2/.test('1221'); // false
```

### 命名引用（ES2018）

使用 `\k<name>` 引用命名分组。

```javascript
// 命名反向引用
/(?<char>\w)\k<char>/.test('aa'); // true
/(?<char>\w)\k<char>/.test('ab'); // false

// 匹配 HTML 标签对
const namedTagPattern = /<(?<tag>\w+)>.*?<\/\k<tag>>/;
namedTagPattern.test('<div>content</div>'); // true

// 混合使用数字和命名引用
/(?<digit>\d)\k<digit>\1/.test('111'); // true
```

### 在 replace 中使用引用

```javascript
// 使用 $1, $2 等引用
'hello world'.replace(/(\w+) (\w+)/, '$2 $1');
// 'world hello'

// 交换相邻字符
'ab cd ef'.replace(/(\w)(\w)/g, '$2$1');
// 'ba dc fe'

// 格式化电话号码
'01012345678'.replace(/(\d{3})(\d{4})(\d{4})/, '$1-$2-$3');
// '010-1234-5678'

// 重复单词检测
const text = 'the the quick fox fox';
text.replace(/\b(\w+) \1\b/g, '$1');
// 'the quick fox'（去除重复单词）
```

## 选择符

选择符 `|` 用于匹配多个可选模式之一。

### 基本用法

```javascript
// 简单选择
/cat|dog/.test('cat'); // true
/cat|dog/.test('dog'); // true
/cat|dog/.test('bird'); // false

// 匹配第一个成功的
'cat'.match(/cat|dog|bird/); // ['cat']
'dog'.match(/cat|dog|bird/); // ['dog']

// 全局匹配
'cat dog bird'.match(/cat|dog|bird/g);
// ['cat', 'dog', 'bird']
```

### 选择符的优先级

```javascript
// 注意：| 的优先级很低
/ab|cd/.test('ab'); // true
/ab|cd/.test('cd'); // true

// 这不是匹配 'abd' 或 'acd'！
// 而是匹配 'ab' 或 'cd'

// 如果要匹配 'abd' 或 'acd'：
/a(b|c)d/.test('abd'); // true
/a(b|c)d/.test('acd'); // true
```

### 分组中的选择

```javascript
// 匹配颜色
const color = /red|green|blue/;
color.test('red');   // true
color.test('green'); // true

// 组合模式
/(red|blue) car/.test('red car');  // true
/(red|blue) car/.test('blue car'); // true
/(red|blue) car/.test('green car'); // false

// 复杂选择
const protocol = /https?|ftp|file/;
protocol.test('http');  // true
protocol.test('https'); // true
protocol.test('ftp');   // true

// 匹配常见文件扩展名
const imageFile = /\.(jpe?g|png|gif|webp|svg)$/i;
imageFile.test('photo.jpg');  // true
imageFile.test('photo.jpeg'); // true
imageFile.test('photo.png');  // true
```

## 先行断言与后行断言

断言用于匹配位置而非字符。

### 先行断言（Lookahead）

```javascript
// 正向先行断言：(?=...)
// 匹配后面是...的位置
/\d+(?=px)/.exec('100px'); // ['100']
/\d+(?=px)/.exec('100em'); // null

// 匹配后面是美元符号的数字
/\d+(?=\$)/.exec('100$'); // ['100']

// 负向先行断言：(?!...)
// 匹配后面不是...的位置
/\d+(?!px)/.exec('100em'); // ['100']
/\d+(?!px)/.exec('100px'); // null

// 验证密码（必须包含数字和大写字母）
const password = /^(?=.*[A-Z])(?=.*\d).{8,}$/;
password.test('Password1'); // true
password.test('password1'); // false（无大写）
password.test('Password');  // false（无数字）
```

### 后行断言（ES2018）

```javascript
// 正向后行断言：(?<=...)
// 匹配前面是...的位置
/(?<=\$)\d+/.exec('$100'); // ['100']
/(?<=\$)\d+/.exec('€100'); // null

// 匹配前面是人民币符号的金额
/(?<=￥)\d+/.exec('￥100'); // ['100']

// 负向后行断言：(?<!...)
// 匹配前面不是...的位置
/(?<!\$)\d+/.exec('€100'); // ['100']
/(?<!\$)\d+/.exec('$100'); // null

// 替换货币符号后的数字
'Price: $100, €200'.replace(/(?<=\$)\d+/g, 'XXX');
// 'Price: $XXX, €200'
```

### 组合使用

```javascript
// 匹配 HTML 标签内容
/(?<=<title>).*?(?=<\/title>)/.exec('<title>Page Title</title>');
// ['Page Title']

// 匹配两个单词之间的内容
/(?<=hello\s)\w+(?=\sworld)/.exec('hello beautiful world');
// ['beautiful']

// 验证：不以 0 开头的数字
/^(?!0)\d+$/.test('123'); // true
/^(?!0)\d+$/.test('012'); // false
```

## 实用案例

### 提取数据

```javascript
// 提取 URL 组成部分
const urlRegex = /^(?<protocol>https?):\/\/(?<host>[^/]+)(?<path>\/.*)?$/;
const urlMatch = urlRegex.exec('https://example.com/path/to/page');
console.log(urlMatch.groups);
// { protocol: 'https', host: 'example.com', path: '/path/to/page' }

// 提取邮箱信息
const emailRegex = /(?<name>[^@]+)@(?<domain>[^.]+)\.(?<tld>\w+)/;
const emailMatch = emailRegex.exec('user@example.com');
console.log(emailMatch.groups);
// { name: 'user', domain: 'example', tld: 'com' }

// 提取 Markdown 链接
const mdLink = /\[(?<text>[^\]]+)\]\((?<url>[^)]+)\)/g;
const markdown = '[Google](https://google.com) [GitHub](https://github.com)';
[...markdown.matchAll(mdLink)].forEach(m => {
  console.log(m.groups);
});
// { text: 'Google', url: 'https://google.com' }
// { text: 'GitHub', url: 'https://github.com' }
```

### 格式验证

```javascript
// 验证手机号（中国大陆）
const phoneRegex = /^1[3-9]\d{9}$/;
phoneRegex.test('13812345678'); // true

// 验证身份证号
const idCardRegex = /^\d{17}[\dXx]$/;
idCardRegex.test('123456789012345678'); // true

// 验证 IPv4 地址
const ipv4Regex = /^(?:(?:25[0-5]|2[0-4]\d|[01]?\d\d?)\.){3}(?:25[0-5]|2[0-4]\d|[01]?\d\d?)$/;
ipv4Regex.test('192.168.1.1'); // true
ipv4Regex.test('256.1.1.1');   // false

// 验证日期格式 YYYY-MM-DD
const dateRegex = /^\d{4}-(?:0[1-9]|1[0-2])-(?:0[1-9]|[12]\d|3[01])$/;
dateRegex.test('2024-01-15'); // true
dateRegex.test('2024-13-15'); // false
```

### 文本处理

```javascript
// 去除 HTML 标签
const html = '<p>Hello <strong>World</strong></p>';
html.replace(/<[^>]+>/g, ''); // 'Hello World'

// 格式化数字（千分位）
'1234567890'.replace(/\B(?=(\d{3})+(?!\d))/g, ',');
// '1,234,567,890'

// 提取所有单词
const text = 'Hello, World! 123';
text.match(/\b[a-zA-Z]+\b/g); // ['Hello', 'World']

// 查找重复单词
const text = 'the the quick fox fox jumps';
text.match(/\b(\w+)\s+\1\b/g); // ['the the', 'fox fox']
```

## 小结

| 特性 | 语法 | 说明 |
|------|------|------|
| `*` | 0 次或多次 | `{0,}` 的简写 |
| `+` | 1 次或多次 | `{1,}` 的简写 |
| `?` | 0 次或 1 次 | `{0,1}` 的简写 |
| `{n}` | 恰好 n 次 | 精确控制次数 |
| `{n,}` | 至少 n 次 | 最小次数限制 |
| `{n,m}` | n 到 m 次 | 范围控制 |
| `(...)` | 捕获分组 | 提取匹配内容 |
| `(?:...)` | 非捕获分组 | 只分组不捕获 |
| `(?<name>...)` | 命名分组 | ES2018 新增 |
| `\n` | 数字反向引用 | 引用第 n 个分组 |
| `\k<name>` | 命名反向引用 | ES2018 新增 |
| `(?=...)` | 正向先行断言 | 后面是... |
| `(?!...)` | 负向先行断言 | 后面不是... |
| `(?<=...)` | 正向后行断言 | 前面是... |
| `(?<!...)` | 负向后行断言 | 前面不是... |
