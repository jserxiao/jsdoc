# 表格样式

```css
table {
    /* 边框合并 */
    border-collapse: collapse;
    border-collapse: separate;
    
    /* 边框间距 */
    border-spacing: 10px;
    
    /* 空单元格 */
    empty-cells: show;
    empty-cells: hide;
    
    /* 表格布局 */
    table-layout: auto;
    table-layout: fixed; /* 固定布局 */
    
    /* 标题位置 */
    caption-side: top;
    caption-side: bottom;
}
```
