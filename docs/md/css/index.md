```
阶段一：基础核心
CSS 基础概念 → 选择器 → 盒模型 → 布局基础
    ↓
阶段二：样式美化
颜色背景 → 文本样式 → 列表样式 → 表格样式
    ↓
阶段三：现代布局
Flexbox → Grid → 响应式设计 → 多列布局
    ↓
阶段四：进阶技巧
变形动画 → 特殊效果 → CSS变量 → 组织架构
    ↓
阶段五：前沿技术
CSS Houdini → 未来特性
```

## 📖 核心概念速查

### 布局三大机制

| 机制 | 特点 | 使用场景 |
|------|------|----------|
| **文档流** | 默认布局 | 基础排版 |
| **浮动** | 脱离文档流，水平排列 | 多栏布局、图文环绕 |
| **定位** | 脱离文档流，层叠显示 | 弹窗、固定头部、叠加效果 |

### 格式化上下文

| 类型 | 触发条件 | 特点 |
|------|----------|------|
| **BFC** | float/position/overflow/flow-root | 独立布局环境，包含浮动 |
| **IFC** | 行内元素容器 | 行内盒子水平排列 |
| **FFC** | display: flex | Flex 子元素布局 |
| **GFC** | display: grid | Grid 子元素布局 |

### 盒模型类型

```css
/* 标准盒模型 */
box-sizing: content-box;
/* width = 内容宽度 */

/* IE 盒模型（推荐） */
box-sizing: border-box;
/* width = 内容 + padding + border */
```

### 定位类型

| 定位 | 脱离文档流 | 定位参考 | 使用场景 |
|------|------------|----------|----------|
| static | 否 | - | 默认 |
| relative | 否 | 自身原位置 | 微调、定位祖先 |
| absolute | 是 | 定位祖先 | 弹窗、图标 |
| fixed | 是 | 视口 | 固定头部 |
| sticky | 否* | 滚动容器 | 吸顶效果 |
