# 数组方法

## 增删改方法

```javascript
const arr = [1, 2, 3];

// 末尾操作
arr.push(4);     // 返回新长度，arr = [1,2,3,4]
arr.pop();       // 返回删除元素，arr = [1,2,3]

// 开头操作
arr.unshift(0);  // 返回新长度，arr = [0,1,2,3]
arr.shift();     // 返回删除元素，arr = [1,2,3]

// splice - 万能方法
arr.splice(1, 1);        // 删除：[1,3]
arr.splice(1, 0, 2);     // 插入：[1,2,3]
arr.splice(1, 1, 'a');   // 替换：[1,'a',3]

// ES2023 非破坏性方法
const newArr = arr.toSorted();
const newArr2 = arr.toReversed();
const newArr3 = arr.toSpliced(1, 1);
const newArr4 = arr.with(0, 'new'); // 替换指定索引
```

## 查找方法

```javascript
const arr = [1, 2, 3, 4, 5, 3];

arr.indexOf(3);        // 2
arr.lastIndexOf(3);    // 5
arr.includes(3);       // true

arr.find(x => x > 3);  // 4（第一个满足的元素）
arr.findIndex(x => x > 3); // 3

// ES2023
arr.findLast(x => x > 3);     // 5
arr.findLastIndex(x => x > 3); // 5

arr.at(0);   // 1（支持负索引）
arr.at(-1);  // 3
```

## 遍历方法

```javascript
const arr = [1, 2, 3, 4, 5];

// forEach - 无返回值
arr.forEach((val, i, arr) => console.log(val));

// map - 返回新数组
const doubled = arr.map(x => x * 2);

// filter - 过滤
const evens = arr.filter(x => x % 2 === 0);

// some / every
arr.some(x => x > 3);   // true（有一个满足）
arr.every(x => x > 0);  // true（全部满足）

// reduce / reduceRight
const sum = arr.reduce((acc, x) => acc + x, 0);
const max = arr.reduce((a, b) => Math.max(a, b));
```

## 排序与反转

```javascript
const arr = [3, 1, 4, 1, 5];

arr.sort();              // [1, 1, 3, 4, 5]
arr.sort((a, b) => b - a); // [5, 4, 3, 1, 1]（降序）
arr.reverse();           // 反转

// 对象排序
const users = [
    { name: 'John', age: 30 },
    { name: 'Jane', age: 25 }
];
users.sort((a, b) => a.age - b.age);
```

## 扁平化与去重

```javascript
// flat / flatMap
const arr = [1, [2, 3], [4, [5, 6]]];
arr.flat();    // [1, 2, 3, 4, [5, 6]]
arr.flat(2);   // [1, 2, 3, 4, 5, 6]
arr.flat(Infinity); // 完全扁平

// flatMap = map + flat
const sentences = ['Hello World', 'Hi There'];
const words = sentences.flatMap(s => s.split(' '));
// ['Hello', 'World', 'Hi', 'There']

// 去重
const unique = [...new Set(arr)];
const unique2 = arr.filter((v, i, a) => a.indexOf(v) === i);
```
