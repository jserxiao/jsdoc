# 包管理工具

## npm

```bash
# 初始化
npm init
npm init -y

# 安装
npm install package
npm install package@version
npm install package --save-dev

# 卸载
npm uninstall package

# 更新
npm update

# 脚本
npm run build
npm test

# 发布
npm publish
```

---

## package.json

```json
{
    "name": "my-package",
    "version": "1.0.0",
    "description": "My package",
    "main": "index.js",
    "scripts": {
        "start": "node index.js",
        "build": "webpack",
        "test": "jest"
    },
    "dependencies": {
        "lodash": "^4.17.21"
    },
    "devDependencies": {
        "webpack": "^5.0.0"
    },
    "engines": {
        "node": ">=16"
    }
}
```

## 语义化版本

```
major.minor.patch

^1.2.3  -> >=1.2.3 <2.0.0
~1.2.3  -> >=1.2.3 <1.3.0
1.2.x   -> >=1.2.0 <1.3.0
```
