# 对象操作

## 对象创建

```javascript
// 对象字面量
const obj = { name: 'John', age: 30 };

// new Object()
const obj2 = new Object();
obj2.name = 'Jane';

// Object.create()
const proto = { greet() { console.log('Hello'); } };
const obj3 = Object.create(proto);
```

---

## 属性描述符

```javascript
const obj = {};

Object.defineProperty(obj, 'name', {
    value: 'John',
    writable: false,      // 不可写
    enumerable: true,     // 可枚举
    configurable: false   // 不可配置
});

Object.getOwnPropertyDescriptor(obj, 'name');
```

---

## getter 与 setter

```javascript
const obj = {
    _name: 'John',
    get name() { return this._name; },
    set name(value) {
        if (value.length > 0) {
            this._name = value;
        }
    }
};
```

---

## 对象遍历

```javascript
const obj = { a: 1, b: 2, c: 3 };

// 遍历键
Object.keys(obj);      // ['a', 'b', 'c']

// 遍历值
Object.values(obj);    // [1, 2, 3]

// 遍历键值对
Object.entries(obj);   // [['a', 1], ['b', 2], ['c', 3]]

// for...in（包含继承的属性）
for (const key in obj) {
    console.log(key, obj[key]);
}
```

---

## 对象冻结与密封

```javascript
const obj = { a: 1 };

// Object.freeze - 完全冻结
Object.freeze(obj);
obj.a = 2;  // 无效

// Object.seal - 密封（不能添加/删除，但可修改）
Object.seal(obj);
obj.a = 2;  // 有效
obj.b = 3;  // 无效

// 判断方法
Object.isFrozen(obj);
Object.isSealed(obj);
```
