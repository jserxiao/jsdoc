# 量词与分组

## 量词

```javascript
/a*/    // 0 次或多次
/a+/    // 1 次或多次
/a?/    // 0 次或 1 次
/a{3}/  // 恰好 3 次
/a{3,}/ // 至少 3 次
/a{3,5}/ // 3 到 5 次

// 贪婪 vs 非贪婪
/a+/    // 贪婪：尽可能多匹配
/a+?/   // 非贪婪：尽可能少匹配

const str = 'aaa';
/a+/.exec(str);  // ['aaa']
/a+?/.exec(str); // ['a']
```

---

## 分组

```javascript
// 捕获分组
/(ab)+/.exec('ababab'); // ['ababab', 'ab']

// 非捕获分组
/(?:ab)+/.exec('ababab'); // ['ababab']

// 命名分组（ES2018）
const regex = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const result = regex.exec('2024-01-15');
console.log(result.groups); // { year: '2024', month: '01', day: '15' }
```

---

## 反向引用

```javascript
// 数字引用
/(\w)\1/.test('aa');  // true（\1 引用第一个分组）
/(\w)(\w)\2\1/.test('abba'); // true

// 命名引用（ES2018）
/(?<char>\w)\k<char>/.test('aa'); // true
```

---

## 选择符

```javascript
/cat|dog/.test('cat'); // true
/cat|dog/.test('dog'); // true

// 组合
/(red|blue) car/.test('red car'); // true
```
