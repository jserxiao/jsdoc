# 正则基础

## 创建正则表达式

```javascript
// 字面量
const regex1 = /pattern/gi;

// RegExp 构造函数
const regex2 = new RegExp('pattern', 'gi');
const regex3 = new RegExp(/pattern/, 'gi'); // ES6

// 动态创建
const str = 'dynamic';
const regex = new RegExp(str, 'g');
```

---

## 元字符

| 字符 | 含义 |
|------|------|
| `.` | 除换行外的任意字符 |
| `\d` | 数字 [0-9] |
| `\D` | 非数字 |
| `\w` | 单词字符 [a-zA-Z0-9_] |
| `\W` | 非单词字符 |
| `\s` | 空白字符（空格、制表、换行等） |
| `\S` | 非空白字符 |
| `\b` | 单词边界 |
| `\B` | 非单词边界 |

---

## 修饰符（flags）

```javascript
/pattern/g  // global - 全局匹配
/pattern/i  // ignoreCase - 忽略大小写
/pattern/m  // multiline - 多行模式
/pattern/s  // dotAll - . 匹配换行
/pattern/u  // unicode - Unicode 模式
/pattern/y  // sticky - 粘连模式

// 组合使用
/pattern/gimsuy
```

---

## 边界符

```javascript
/^hello/    // 匹配以 hello 开头
/world$/    // 匹配以 world 结尾
/^hello$/   // 精确匹配 hello

// 多行模式下的 ^
/^hello/gm  // 每行开头匹配
```

---

## 字符集合

```javascript
/[abc]/     // a、b、c 任意一个
/[a-z]/     // 小写字母
/[A-Z]/     // 大写字母
/[0-9]/     // 数字
/[a-zA-Z0-9]/ // 字母和数字

// 排除
/[^abc]/    // 除 a、b、c 外任意字符

// 常用等价
/\d/ === /[0-9]/
/\w/ === /[a-zA-Z0-9_]/
/\s/ === /[ \t\r\n\f]/
```
