# 常用正则模式

## 数字匹配

```javascript
// 整数
/^-?\d+$/

// 正整数
/^[1-9]\d*$/

// 浮点数
/^-?\d+\.\d+$/

// 数字（整数或浮点数）
/^-?\d+(\.\d+)?$/

// 科学计数法
/^-?\d+(\.\d+)?([eE][+-]?\d+)?$/
```

---

## 验证模式

```javascript
// 手机号（中国）
/^1[3-9]\d{9}$/

// 邮箱
/^[\w-]+(\.[\w-]+)*@[\w-]+(\.[\w-]+)+$/

// 身份证（18位）
/^\d{17}[\dXx]$/

// URL
/^https?:\/\/[\w\-]+(\.[\w\-]+)+[/#?]?.*$/

// IPv4
/^(\d{1,3}\.){3}\d{1,3}$/

// 邮编（中国）
/^\d{6}$/

// 中文
/^[\u4e00-\u9fa5]+$/

// 密码（8-20位，包含字母和数字）
/^(?=.*[a-zA-Z])(?=.*\d)[a-zA-Z\d]{8,20}$/
```

---

## 提取模式

```javascript
// 提取 HTML 标签
/<(\w+)[^>]*>.*?<\/\1>/gs

// 提取属性值
/attr="([^"]*)"/

// 提取 URL 参数
/[?&](\w+)=([^&]*)/g

// 提取日期
/(\d{4})-(\d{2})-(\d{2})/
```

---

## 替换模式

```javascript
// 移除 HTML 标签
str.replace(/<[^>]*>/g, '')

// 移除空白
str.replace(/\s+/g, ' ').trim()

// 千分位格式化
'1234567'.replace(/\B(?=(\d{3})+(?!\d))/g, ',') // '1,234,567'

// 隐藏手机号中间四位
'13812345678'.replace(/(\d{3})\d{4}(\d{4})/, '$1****$2')
// '138****5678'
```
