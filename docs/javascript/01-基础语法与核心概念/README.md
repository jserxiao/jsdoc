## 📖 核心概念速查

### 变量声明三原则

```
1. 默认使用 const — 大多数变量不需要重新赋值
2. 需要重新赋值时用 let — 如循环计数器
3. 避免使用 var — 它的问题太多
```

### 数据类型分类

```
原始类型（7种）         引用类型
├── string             ├── object
├── number             ├── array
├── boolean            ├── function
├── null               ├── map
├── undefined          ├── set
├── symbol (ES6)       └── ...
└── bigint (ES2020)
```

### 常见陷阱提醒

⚠️ **作用域陷阱**：`var` 是函数作用域，会泄漏到块外
⚠️ **提升陷阱**：`var` 声明会提升，可能导致 undefined
⚠️ **TDZ 陷阱**：`let/const` 在声明前访问会报错
⚠️ **类型陷阱**：`typeof null === 'object'`（历史 bug）
