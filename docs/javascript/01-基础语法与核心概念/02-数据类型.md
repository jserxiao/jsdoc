# 数据类型

> JavaScript 是一门动态类型语言，理解数据类型是编写可靠代码的基础。

## 学习要点

- 📦 区分原始类型和引用类型
- 🔍 掌握类型检测方法
- 🔄 理解类型转换规则
- 📋 学会深浅拷贝的实现

---

## 1. 原始类型（Primitive Types）

原始类型的值是**不可变**的，存储在**栈内存**中。

### 七种原始类型

| 类型 | 说明 | 示例 |
|------|------|------|
| String | 字符串 | `'hello'` |
| Number | 数字 | `42`, `3.14` |
| Boolean | 布尔值 | `true`, `false` |
| null | 空值 | `null` |
| undefined | 未定义 | `undefined` |
| Symbol | 符号（ES6） | `Symbol('id')` |
| BigInt | 大整数（ES2020） | `9007199254740991n` |

### String（字符串）

```javascript
// 三种定义方式
let str1 = 'Hello';       // 单引号
let str2 = "World";       // 双引号
let str3 = `Hello ${str2}`; // 模板字符串（推荐）

// 字符串是不可变的
let str = 'hello';
str[0] = 'H';  // 无效，不会改变
console.log(str); // 'hello'

// 常用方法
'hello'.length;           // 5
'hello'.toUpperCase();    // 'HELLO'
'hello'.indexOf('l');     // 2
'hello'.slice(1, 3);      // 'el'
'hello'.split('');        // ['h','e','l','l','o']
```

### Number（数字）

```javascript
// 整数和浮点数
let integer = 42;
let float = 3.14;
let scientific = 2.5e6;  // 2500000

// 特殊值
let notANumber = NaN;    // 不是一个数字
let infinity = Infinity; // 无穷大

// 检测 NaN
console.log(NaN === NaN);      // false
console.log(Number.isNaN(NaN)); // true（推荐）

// 精度问题
console.log(0.1 + 0.2);  // 0.30000000000000004
console.log(0.1 + 0.2 === 0.3); // false

// 解决精度问题
function isEqual(a, b) {
    return Math.abs(a - b) < Number.EPSILON;
}
```

### Boolean（布尔值）

```javascript
let isTrue = true;
let isFalse = false;

// 假值（Falsy）- 转为布尔值时为 false
// 只有这 8 个值：
// false, 0, -0, 0n, '', null, undefined, NaN

// 真值（Truthy）- 除了假值都是真值
// 如：1, 'hello', [], {}, function(){}, -1, '0', 'false'
```

### null 和 undefined

```javascript
// null：表示"有值，但值为空"
let data = null; // 主动设置为空

// undefined：表示"尚未赋值"
let data2;       // 自动为 undefined

// 检测
console.log(typeof null);      // 'object' ⚠️（历史遗留 bug）
console.log(typeof undefined); // 'undefined'

// 比较
console.log(null == undefined);  // true
console.log(null === undefined); // false（推荐用严格相等）
```

### Symbol（ES6）

```javascript
// 每个 Symbol 都是唯一的
let sym1 = Symbol('description');
let sym2 = Symbol('description');
console.log(sym1 === sym2); // false

// 常见用途：创建私有属性
const ID = Symbol('id');
const user = {
    name: 'John',
    [ID]: 123  // Symbol 作为键
};
console.log(user[ID]); // 123
console.log(Object.keys(user)); // ['name']（Symbol 不会被枚举）
```

### BigInt（ES2020）

```javascript
// 用于表示大整数（超过 Number 的安全范围）
let bigNum = 9007199254740991n;
let bigNum2 = BigInt('9007199254740991');

// BigInt 与 Number 不能混合运算
console.log(1n + 2n);    // 3n ✅
// console.log(1n + 2);  // TypeError ❌

// 需要显式转换
console.log(1n + BigInt(2)); // 3n
console.log(Number(1n) + 2); // 3
```

## 2. 引用类型（Reference Types）

引用类型的值是**可变**的，存储在**堆内存**中，变量保存的是**引用地址**。

### 存储对比

```
原始类型：
┌─────────────┐
│ 变量 a = 1  │ ← 值直接存储
└─────────────┘

引用类型：
┌─────────────┐      ┌─────────────────────┐
│ 变量 obj    │ ───→ │ { name: 'John' }    │
│ (存储地址)   │      │ (堆内存中的对象)     │
└─────────────┘      └─────────────────────┘
```

### Object（对象）

```javascript
const obj = {
    name: 'John',
    age: 30,
    greet() {
        console.log(`Hello, I'm ${this.name}`);
    }
};

// 访问属性
console.log(obj.name);      // 'John'
console.log(obj['name']);   // 'John'（动态访问）

// 添加/修改属性
obj.email = 'john@example.com';
obj.age = 31;

// 删除属性
delete obj.email;
```

### Array（数组）

```javascript
const arr = [1, 2, 3, 'four', { five: 5 }];

// 访问元素
console.log(arr[0]);  // 1
console.log(arr.length); // 5

// 常用方法
arr.push(6);        // 末尾添加 → [1,2,3,'four',{five:5},6]
arr.pop();          // 末尾移除 → [1,2,3,'four',{five:5}]
arr.unshift(0);     // 开头添加 → [0,1,2,3,'four',{five:5}]
arr.shift();        // 开头移除 → [1,2,3,'four',{five:5}]
```

### Function（函数）

```javascript
// 函数也是对象
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(typeof greet);    // 'function'
console.log(greet.name);      // 'greet'
console.log(greet.length);    // 1（参数个数）
```

## 3. 类型检测

### typeof 操作符

```javascript
typeof 'hello'     // 'string'
typeof 42          // 'number'
typeof true        // 'boolean'
typeof undefined   // 'undefined'
typeof null        // 'object' ⚠️（历史 bug）
typeof {}          // 'object'
typeof []          // 'object'
typeof (() => {})  // 'function'
typeof Symbol()    // 'symbol'
typeof 1n          // 'bigint'
```

### 检测 null

```javascript
// 正确检测 null
value === null

// 检测 null 或 undefined
value == null
```

### 检测数组

```javascript
const arr = [];

// 方法 1：Array.isArray（推荐）
Array.isArray(arr); // true

// 方法 2：instanceof
arr instanceof Array; // true

// 方法 3：Object.prototype.toString
Object.prototype.toString.call(arr) === '[object Array]';
```

### 精确类型检测

```javascript
function getType(value) {
    return Object.prototype.toString.call(value).slice(8, -1);
}

getType(null);       // 'Null'
getType(undefined);  // 'Undefined'
getType([]);         // 'Array'
getType({});         // 'Object'
getType(new Date()); // 'Date'
getType(/regex/);    // 'RegExp'
```

## 4. 类型转换

### 显式转换

```javascript
// 转字符串
String(123);        // '123'
(123).toString();   // '123'
123 + '';           // '123'

// 转数字
Number('123');      // 123
parseInt('123.45'); // 123
parseFloat('123.45'); // 123.45
+'123';             // 123

// 转布尔值
Boolean(1);         // true
!!1;                // true
```

### 隐式转换陷阱

```javascript
// 字符串拼接（+ 运算符）
'1' + 2;       // '12'
'1' + true;    // '1true'

// 数学运算（其他运算符）
'5' - 2;       // 3
'5' * '2';     // 10

// 比较运算
'5' == 5;      // true（类型转换后比较）
'5' === 5;     // false（严格比较，推荐）

// ⚠️ 奇葩案例
[] == false;   // true
[] == ![];     // true
'' == 0;       // true
```

## 5. 深拷贝与浅拷贝

### 浅拷贝

```javascript
const original = { a: 1, b: { c: 2 } };

// 方法 1：Object.assign
const copy1 = Object.assign({}, original);

// 方法 2：展开运算符
const copy2 = { ...original };

// 问题：嵌套对象仍是引用
copy1.b.c = 999;
console.log(original.b.c); // 999（原对象被修改）
```

### 深拷贝

```javascript
// 方法 1：JSON（简单但有局限）
const deepCopy1 = JSON.parse(JSON.stringify(original));
// 缺点：无法处理函数、Symbol、undefined、循环引用

// 方法 2：structuredClone（现代浏览器）
const deepCopy2 = structuredClone(original);
// 支持循环引用，但仍不支持函数

// 方法 3：手动实现
function deepClone(obj, hash = new WeakMap()) {
    if (obj === null || typeof obj !== 'object') return obj;
    if (hash.has(obj)) return hash.get(obj); // 处理循环引用
    
    const clone = Array.isArray(obj) ? [] : {};
    hash.set(obj, clone);
    
    for (const key in obj) {
        if (obj.hasOwnProperty(key)) {
            clone[key] = deepClone(obj[key], hash);
        }
    }
    return clone;
}
```
