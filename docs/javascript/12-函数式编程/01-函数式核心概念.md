# 函数式核心概念

## 纯函数

```javascript
// 纯函数：相同输入总是返回相同输出，无副作用
function add(a, b) {
    return a + b;
}

// 非纯函数：依赖外部状态
let counter = 0;
function increment() {
    return ++counter; // 有副作用
}

// 非纯函数：修改参数
function addItem(arr, item) {
    arr.push(item); // 修改原数组
    return arr;
}

// 纯函数版本
function addItemPure(arr, item) {
    return [...arr, item]; // 返回新数组
}
```

## 不可变性

```javascript
// 可变操作（避免）
const arr = [1, 2, 3];
arr.push(4); // 修改原数组

// 不可变操作（推荐）
const newArr = [...arr, 4];
const newArr2 = arr.concat(4);
const newArr3 = [...arr.slice(0, 1), 'new', ...arr.slice(2)];

// 对象不可变
const obj = { a: 1, b: 2 };
const newObj = { ...obj, b: 3 }; // 更新 b
const newObj2 = { ...obj, c: 3 }; // 添加 c
const { a, ...rest } = obj;       // 删除 a
```

## 高阶函数

```javascript
// 接收函数作为参数
function map(arr, fn) {
    return arr.reduce((acc, item) => [...acc, fn(item)], []);
}

// 返回函数
function multiplier(factor) {
    return x => x * factor;
}

const double = multiplier(2);
console.log(double(5)); // 10
```
