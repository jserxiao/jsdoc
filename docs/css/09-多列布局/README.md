# 九、多列布局

```css
.multi-column {
    /* 列数 */
    column-count: 3;
    
    /* 列宽 */
    column-width: 200px;
    
    /* 间距 */
    column-gap: 20px;
    
    /* 分隔线 */
    column-rule: 1px solid #ccc;
    
    /* 跨列 */
    column-span: all;
}

h2 {
    column-span: all; /* 标题跨所有列 */
}
```
