# 类的基本语法

## 概述

TypeScript 扩展了 JavaScript 类,增加了类型和访问修饰符。

## 一、类定义

```typescript
class User {
  // 属性
  name: string
  age: number

  // 构造函数
  constructor(name: string, age: number) {
    this.name = name
    this.age = age
  }

  // 方法
  greet(): string {
    return `Hello, I'm ${this.name}`
  }
}

const user = new User('Alice', 25)
console.log(user.greet())
```

## 二、访问修饰符

### public(默认)

```typescript
class User {
  public name: string = 'Alice'
}

const user = new User()
console.log(user.name)  // ✅ 可以访问
```

### private

```typescript
class User {
  private age: number = 25

  getAge(): number {
    return this.age  // ✅ 类内部可以访问
  }
}

const user = new User()
// console.log(user.age)  // ❌ 不能访问
```

### protected

```typescript
class Animal {
  protected name: string = 'Animal'
}

class Dog extends Animal {
  bark(): void {
    console.log(this.name)  // ✅ 子类可以访问
  }
}

const dog = new Dog()
// console.log(dog.name)  // ❌ 不能访问
```

## 三、readonly

```typescript
class User {
  readonly id: number

  constructor(id: number) {
    this.id = id
  }
}

const user = new User(1)
// user.id = 2  // ❌ 不能修改
```

## 四、参数属性

```typescript
class User {
  constructor(
    public name: string,
    private age: number,
    readonly id: number
  ) {}
}

const user = new User('Alice', 25, 1)
console.log(user.name)  // Alice
console.log(user.id)    // 1
```

## 五、存取器

```typescript
class User {
  private _age: number = 0

  get age(): number {
    return this._age
  }

  set age(value: number) {
    if (value > 0) {
      this._age = value
    }
  }
}

const user = new User()
user.age = 25
console.log(user.age)  // 25
```

## 六、静态成员

```typescript
class User {
  static count: number = 0

  constructor(public name: string) {
    User.count++
  }

  static getCount(): number {
    return User.count
  }
}

const user1 = new User('Alice')
const user2 = new User('Bob')
console.log(User.getCount())  // 2
```

## 七、抽象类

```typescript
abstract class Animal {
  abstract makeSound(): void

  move(): void {
    console.log('Moving...')
  }
}

class Dog extends Animal {
  makeSound(): void {
    console.log('Woof!')
  }
}

const dog = new Dog()
dog.makeSound()  // Woof!
dog.move()       // Moving...
```

## 八、总结

- public/private/protected 控制访问
- readonly 修饰只读属性
- get/set 实现存取器
- static 定义静态成员
- abstract 定义抽象类
