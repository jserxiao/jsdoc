# 映射类型

## 概述

映射类型基于旧类型创建新类型。

## 一、基本语法

```typescript
type Readonly<T> = {
  readonly [P in keyof T]: T[P]
}

interface User {
  name: string
  age: number
}

type ReadonlyUser = Readonly<User>
// { readonly name: string; readonly age: number }
```

## 二、常见映射类型

```typescript
// Partial
type MyPartial<T> = {
  [P in keyof T]?: T[P]
}

// Required
type MyRequired<T> = {
  [P in keyof T]-?: T[P]
}

// Pick
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P]
}

// Record
type MyRecord<K extends keyof any, V> = {
  [P in K]: V
}
```

## 三、修饰符

```typescript
// 添加 readonly
type Readonly<T> = {
  readonly [P in keyof T]: T[P]
}

// 移除 readonly
type Mutable<T> = {
  -readonly [P in keyof T]: T[P]
}

// 添加可选
type Optional<T> = {
  [P in keyof T]?: T[P]
}

// 移除可选
type Required<T> = {
  [P in keyof T]-?: T[P]
}
```

## 四、实战示例

```typescript
// 获取函数返回值类型
type ReturnType<T> = T extends (...args: any[]) => infer R ? R : never

function getUser() {
  return { id: 1, name: 'Alice' }
}

type User = ReturnType<typeof getUser>
// { id: number; name: string }
```

## 五、总结

- `[P in keyof T]`: 遍历属性
- `readonly`, `?`: 添加修饰符
- `-readonly`, `-?`: 移除修饰符
- `infer`: 类型推断
