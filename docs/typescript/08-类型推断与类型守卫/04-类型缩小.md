# 类型缩小

## 概述

类型缩小是通过检查将类型从宽泛缩小到具体的过程。

## 一、typeof 缩小

```typescript
function process(value: string | number) {
  if (typeof value === 'string') {
    // value: string
    return value.toUpperCase()
  } else {
    // value: number
    return value * 2
  }
}
```

## 二、真值缩小

```typescript
function process(value: string | null | undefined) {
  if (value) {
    // value: string
    console.log(value.toUpperCase())
  }
}

// 使用 ||
function greet(name: string | null) {
  const displayName = name || 'stranger'
}
```

## 三、等值缩小

```typescript
function process(value: string | number | null) {
  if (value === null) {
    return 'null'
  }

  if (value === 'specific') {
    // value: 'specific'
  }
}
```

## 四、in 操作符缩小

```typescript
interface Bird {
  fly(): void
  layEggs(): void
}

interface Fish {
  swim(): void
  layEggs(): void
}

function process(animal: Bird | Fish) {
  if ('fly' in animal) {
    // animal: Bird
    animal.fly()
  } else {
    // animal: Fish
    animal.swim()
  }
}
```

## 五、instanceof 缩小

```typescript
function process(value: Date | string) {
  if (value instanceof Date) {
    // value: Date
    return value.toISOString()
  } else {
    // value: string
    return value.toUpperCase()
  }
}
```

## 六、赋值缩小

```typescript
let x: string | number

x = 'hello'  // x: string
console.log(x.toUpperCase())

x = 42       // x: number
console.log(x.toFixed(2))
```

## 七、控制流分析

```typescript
function process(value: string | number) {
  if (typeof value === 'string') {
    return value.toUpperCase()
  }

  // value: number (已经排除了 string)
  return value.toFixed(2)
}
```

## 八、总结

- typeof, instanceof, in 操作符缩小
- 真值、等值检查缩小
- 赋值和控制流缩小
- 类型守卫缩小
