# 函数类型定义

## 概述

TypeScript 支持多种方式定义函数类型。

## 一、函数声明

```typescript
function add(a: number, b: number): number {
  return a + b
}
```

## 二、函数表达式

```typescript
const add = function(a: number, b: number): number {
  return a + b
}

// 箭头函数
const multiply = (a: number, b: number): number => a * b
```

## 三、函数类型别名

```typescript
type MathFunction = (a: number, b: number) => number

const add: MathFunction = (a, b) => a + b
const subtract: MathFunction = (a, b) => a - b
```

## 四、接口定义函数

```typescript
interface MathFunction {
  (a: number, b: number): number
}

const divide: MathFunction = (a, b) => a / b
```

## 五、构造函数类型

```typescript
interface Constructor {
  new (name: string): User
}

type Constructor2 = new (name: string) => User

class User {
  constructor(public name: string) {}
}

function create(ctor: Constructor, name: string): User {
  return new ctor(name)
}
```

## 六、函数重载签名

```typescript
function reverse(x: number): number
function reverse(x: string): string
function reverse(x: number | string): number | string {
  if (typeof x === 'number') {
    return Number(x.toString().split('').reverse().join(''))
  } else {
    return x.split('').reverse().join('')
  }
}
```

## 七、实战示例

```typescript
// API 处理函数
type RequestHandler = (
  req: { method: string; path: string; body?: any },
  res: { status: (code: number) => void; json: (data: any) => void }
) => void | Promise<void>

const handler: RequestHandler = async (req, res) => {
  res.status(200)
  res.json({ message: 'OK' })
}

// 事件处理函数
type EventHandler<E = Event> = (event: E) => void

const clickHandler: EventHandler<MouseEvent> = (e) => {
  console.log(e.clientX, e.clientY)
}
```

## 八、总结

- 函数声明: `function name(): Type`
- 函数表达式: `const fn = (): Type => {}`
- 类型别名: `type Fn = (args) => Type`
- 接口定义: `interface Fn { (args): Type }`
