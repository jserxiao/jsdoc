# 剩余参数

## 概述

剩余参数允许函数接受不定数量的参数。

## 一、基本用法

```typescript
function sum(...numbers: number[]): number {
  return numbers.reduce((total, n) => total + n, 0)
}

sum(1, 2, 3)      // 6
sum(1, 2, 3, 4)   // 10
```

## 二、剩余参数类型

```typescript
// 数组类型
function concat(...strings: string[]): string {
  return strings.join('')
}

// 元组类型
function pair(...pair: [string, number]): void {
  console.log(pair[0], pair[1])
}

pair('hello', 42)
```

## 三、剩余参数与普通参数

```typescript
function log(separator: string, ...messages: string[]): void {
  console.log(messages.join(separator))
}

log(',', 'a', 'b', 'c')  // 'a,b,c'
```

## 四、实战示例

```typescript
// 类型安全的事件发射器
class EventEmitter {
  private events = new Map<string, Set<Function>>()

  on<T extends any[]>(event: string, handler: (...args: T) => void): void {
    if (!this.events.has(event)) {
      this.events.set(event, new Set())
    }
    this.events.get(event)!.add(handler)
  }

  emit<T extends any[]>(event: string, ...args: T): void {
    this.events.get(event)?.forEach(handler => handler(...args))
  }
}

const emitter = new EventEmitter()
emitter.on('click', (x: number, y: number) => {
  console.log(`Clicked at (${x}, ${y})`)
})

emitter.emit('click', 100, 200)
```

## 五、总结

- 剩余参数用 `...` 标记
- 必须是最后一个参数
- 类型为数组或元组
