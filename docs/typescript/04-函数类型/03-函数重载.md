# 函数重载

## 概述

函数重载允许一个函数接受不同类型或数量的参数,并提供不同的返回类型。

## 一、基本语法

```typescript
function reverse(x: number): number
function reverse(x: string): string
function reverse(x: number | string): number | string {
  if (typeof x === 'number') {
    return Number(x.toString().split('').reverse().join(''))
  } else {
    return x.split('').reverse().join('')
  }
}

const num = reverse(123)      // number: 321
const str = reverse('hello')  // string: 'olleh'
```

## 二、重载签名顺序

```typescript
function getLength(x: string): number
function getLength(x: any[]): number
function getLength(x: string | any[]): number {
  return x.length
}

// TypeScript 会按顺序匹配,更具体的类型放在前面
```

## 三、实战示例

```typescript
// createElement
function createElement(tag: 'a'): HTMLAnchorElement
function createElement(tag: 'canvas'): HTMLCanvasElement
function createElement(tag: 'table'): HTMLTableElement
function createElement(tag: string): HTMLElement {
  return document.createElement(tag)
}

const anchor = createElement('a')      // HTMLAnchorElement
const canvas = createElement('canvas')  // HTMLCanvasElement
const div = createElement('div')        // HTMLElement
```

## 四、最佳实践

### 1. 实现签名要兼容所有重载

```typescript
// ✅ 正确
function fn(x: string): string
function fn(x: number): number
function fn(x: string | number): string | number {
  return x
}

// ❌ 错误: 实现签名不兼容
function fn2(x: string): string
function fn2(x: number): number
function fn2(x: boolean): boolean  // 错误
```

### 2. 避免过度重载

```typescript
// ❌ 过度重载
function getValue(x: 1): string
function getValue(x: 2): string
function getValue(x: 3): string
// ...

// ✅ 使用联合类型
function getValue(x: number): string {
  return String(x)
}
```

## 五、总结

- 重载提供更精确的类型推断
- 实现签名必须兼容所有重载
- 按从具体到通用的顺序排列
