# 索引签名

## 概述

索引签名允许定义动态属性名的对象类型。

## 一、字符串索引签名

```typescript
interface StringMap {
  [key: string]: string
}

const map: StringMap = {
  name: 'Alice',
  city: 'Beijing'
}

map.country = 'China'  // ✅
```

## 二、数字索引签名

```typescript
interface StringArray {
  [index: number]: string
}

const arr: StringArray = ['a', 'b', 'c']
console.log(arr[0])  // 'a'
```

## 三、混合索引

```typescript
interface MixedMap {
  [key: string]: string | number
  [index: number]: string
  length: number  // 必须符合索引签名
}
```

## 四、只读索引签名

```typescript
interface ReadonlyMap {
  readonly [key: string]: string
}

const map: ReadonlyMap = {
  name: 'Alice'
}

// map.name = 'Bob'  // ❌ 只读
```

## 五、模板字面量索引 (TS 4.4+)

```typescript
interface Env {
  [key: `${string}_URL`]: string
  [key: `${string}_PORT`]: number
}

const env: Env = {
  API_URL: 'https://api.example.com',
  API_PORT: 3000
}
```

## 六、限制与注意

```typescript
// ❌ 数字索引返回类型必须是字符串索引返回类型的子类型
interface Bad {
  [key: string]: string
  [index: number]: number  // 编译错误
}

// ✅ 正确
interface Good {
  [key: string]: string | number
  [index: number]: string
}
```

## 七、实战示例

```typescript
// 配置对象
interface Config {
  [key: string]: any
  apiUrl: string
  timeout: number
}

// 事件映射
interface EventMap {
  [event: string]: (...args: any[]) => void
}

class EventEmitter {
  private events: EventMap = {}

  on(event: string, handler: (...args: any[]) => void) {
    this.events[event] = handler
  }

  emit(event: string, ...args: any[]) {
    this.events[event]?.(...args)
  }
}
```

## 八、总结

- 字符串索引: `[key: string]: Type`
- 数字索引: `[index: number]: Type`
- 只读索引: `readonly [key: string]: Type`
- 索引签名用于动态属性
