# 类型挑战

## 概述

通过解决类型挑战提升类型编程能力。

## 一、Pick

```typescript
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P]
}

// 测试
interface Todo {
  title: string
  description: string
  completed: boolean
}

type TodoPreview = MyPick<Todo, 'title' | 'completed'>
// { title: string; completed: boolean }
```

## 二、Readonly

```typescript
type MyReadonly<T> = {
  readonly [P in keyof T]: T[P]
}

// 测试
interface Todo {
  title: string
}

const todo: MyReadonly<Todo> = {
  title: 'Delete inactive users'
}

// todo.title = 'Hello' // Error
```

## 三、Tuple to Object

```typescript
type TupleToObject<T extends readonly (string | number | symbol)[]> = {
  [P in T[number]]: P
}

// 测试
const tuple = ['tesla', 'model 3', 'model X'] as const
type result = TupleToObject<typeof tuple>
// { tesla: 'tesla'; 'model 3': 'model 3'; 'model X': 'model X' }
```

## 四、First of Array

```typescript
type First<T extends any[]> = T extends [infer F, ...any[]] ? F : never

// 测试
type arr1 = ['a', 'b', 'c']
type arr2 = [3, 2, 1]

type head1 = First<arr1> // 'a'
type head2 = First<arr2> // 3
```

## 五、Length of Tuple

```typescript
type Length<T extends readonly any[]> = T['length']

// 测试
type tesla = ['tesla', 'model 3', 'model X', 'model Y']
type spaceX = ['FALCON 9', 'FALCON HEAVY', 'DRAGON', 'STARSHIP']

type teslaLength = Length<tesla> // 4
type spaceXLength = Length<spaceX> // 4
```

## 六、Exclude

```typescript
type MyExclude<T, U> = T extends U ? never : T

// 测试
type Result = MyExclude<'a' | 'b' | 'c', 'a'> // 'b' | 'c'
```

## 七、总结

- 映射类型: `{ [P in K]: T[P] }`
- 条件类型: `T extends U ? X : Y`
- infer: 类型推断
- 索引访问: `T[keyof T]`
