# 目标与模块

## 概述

配置编译输出目标和模块系统。

## 一、编译目标

```json
{
  "compilerOptions": {
    "target": "ES2020"
  }
}
```

可选值:
- `ES3` (默认)
- `ES5`
- `ES2015` / `ES6`
- `ES2016` / `ES7`
- `ES2017`
- `ES2018`
- `ES2019`
- `ES2020`
- `ES2021`
- `ES2022`
- `ESNext`

## 二、模块系统

```json
{
  "compilerOptions": {
    "module": "ESNext"
  }
}
```

可选值:
- `CommonJS` (Node.js)
- `AMD` (RequireJS)
- `System` (SystemJS)
- `UMD`
- `ES6` / `ES2015`
- `ES2020`
- `ES2022`
- `ESNext`
- `None`

## 三、模块解析

```json
{
  "compilerOptions": {
    "moduleResolution": "node"
  }
}
```

可选值:
- `classic` (旧版)
- `node` (Node.js 风格)
- `node16` / `nodenext`

## 四、库配置

```json
{
  "compilerOptions": {
    "lib": ["ES2020", "DOM", "DOM.Iterable"]
  }
}
```

## 五、输出配置

```json
{
  "compilerOptions": {
    "outDir": "./dist",
    "outFile": "./bundle.js",
    "rootDir": "./src",
    "declaration": true,
    "declarationMap": true,
    "sourceMap": true
  }
}
```

## 六、总结

- `target`: 指定编译目标
- `module`: 指定模块系统
- `moduleResolution`: 指定模块解析策略
- `lib`: 指定运行时库
